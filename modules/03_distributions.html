The foundation for every inference question

Concept

1. Binomial Distribution

The binomial distribution models the number of successes in a fixed number of independent trials, each with the same probability of success.

Conditions (all must hold)

  • Fixed number of trials n
  • Each trial has exactly two outcomes (success/failure)
  • Trials are independent
  • Probability of success p is constant across trials
Binomial PMF (from cheat sheet) P(X = x) = n! / (x! (n − x)!) · px · (1 − p)n − x
Expected Value & Standard Deviation E(X) = n · p      SD(X) = √(n · p · (1 − p))

R Functions

dbinom(x, size, prob) — P(X = x), the exact probability of x successes.
pbinom(x, size, prob) — P(X ≤ x), the cumulative probability up to x.
Worked Example

In a random sample of 25 households, 12% own an electric vehicle. What is P(X = 3)?

Here: n = 25, p = 0.12, x = 3.

Calculation
P(X = 3) = C(25, 3) · 0.123 · 0.8822
= 2300 · 0.001728 · 0.05863
= 0.2330

In R: dbinom(3, 25, 0.12) = 0.2330
Exam tip: "At least one" means P(X ≥ 1) = 1 − P(X = 0). This complement trick appears constantly.
Concept

2. Poisson Distribution

The Poisson distribution models the number of events in a fixed interval (time, area, volume) when events occur independently at a constant average rate.

Conditions

  • Events occur independently
  • The average rate λ is constant over the interval
  • Two events cannot occur at exactly the same instant
Poisson PMF (from cheat sheet) P(X = x) = λx · e−λ / x!
Expected Value & Standard Deviation E(X) = λ      Var(X) = λ      SD(X) = √λ

R Functions

dpois(x, lambda) — P(X = x)
ppois(x, lambda) — P(X ≤ x)
Worked Example

A hospital emergency room receives an average of 6 patients per hour. What is P(X = 4)?

Calculation
P(X = 4) = 64 · e−6 / 4!
= 1296 · 0.002479 / 24
= 0.1339

In R: dpois(4, 6) = 0.1339
Poisson as Binomial approximation: When n is large and p is small, Bin(n, p) ≈ Poisson(λ = np). Rule of thumb: n ≥ 100, p ≤ 0.01.
Concept

3. Geometric Distribution

The geometric distribution models the number of trials until the first success. Each trial is independent with constant success probability p.

Geometric PMF (from cheat sheet) P(X = x) = (1 − p)x − 1 · p     for x ∈ ℕ (i.e., x = 1, 2, 3, ...)
Expected Value & Standard Deviation E(X) = 1/p      SD(X) = √((1 − p) / p2)
Worked Example

A free-throw shooter has a 70% success rate. What is the probability it takes exactly 3 attempts to make the first basket?

Calculation
P(X = 3) = (1 − 0.7)2 · 0.7 = 0.32 · 0.7 = 0.09 · 0.7 = 0.063

In R: dgeom(2, 0.7) = 0.063
Note: R's dgeom counts failures (x−1), not trials.
Watch out: The geometric distribution is memoryless. P(X > s + t | X > s) = P(X > t). Past failures don't affect future probability.
Concept

4. Negative Binomial Distribution

The negative binomial generalizes the geometric: it models the number of trials until the k-th success. Like the geometric, each trial is independent with constant success probability p.

Intuition

Think of it this way: you keep running independent Bernoulli trials (each with success probability p) and you stop as soon as you have accumulated k successes. X counts the total number of trials needed.

Negative Binomial PMF (from cheat sheet) P(X = x) = (x − 1)! / ((k − 1)! (x − k)!) · pk · (1 − p)x − k     for x ≥ k
Reading the formula: The combinatorial term C(x−1, k−1) = (x−1)! / ((k−1)!(x−k)!) counts the ways to arrange k−1 successes in the first x−1 trials (the last trial must be the k-th success). Then pk accounts for k successes and (1−p)x−k for the remaining failures.
Expected Value & Variance E(X) = k/p      Var(X) = k(1 − p) / p2
Relationship: Geometric is just Negative Binomial with k = 1. Setting k = 1 in all the formulas above recovers the geometric distribution exactly.
Worked Example

A recruiter interviews candidates, and each has a 20% chance of being hired. How many interviews until 3 hires, on average?

Calculation
E(X) = k/p = 3 / 0.20 = 15 interviews

SD(X) = √(k(1−p)/p2) = √(3 · 0.8 / 0.04) = √60 = 7.75 interviews
Worked Example 2 — Computing a probability

What is the probability that the recruiter needs exactly 5 interviews to make 3 hires?

Calculation
x = 5, k = 3, p = 0.2
P(X = 5) = C(4, 2) · 0.23 · 0.82
= 6 · 0.008 · 0.64
= 0.03072
R convention warning: R's dnbinom(x, size, prob) parameterizes differently — x counts failures before the k-th success, not total trials. So for the example above: dnbinom(2, size=3, prob=0.2) = 0.03072, where x = 5 − 3 = 2 failures.
Concept

5. Continuous Distributions — PDF, CDF & Empirical CDF

For continuous random variables, we cannot assign probabilities to individual points. Instead we use density functions and compute probabilities as areas under the curve.

5.1 Probability Density Function (PDF)

A function f(x) is a valid PDF if f(x) ≥ 0 for all x and ∫ f(x) dx = 1 over the entire domain.

Probability from a PDF P(a ≤ X ≤ b) = ∫ab f(x) dx
Key distinction: For continuous variables, P(X = x) = 0 for any single value x. Probabilities are always intervals. Consequently, P(a ≤ X ≤ b) = P(a < X < b) — including or excluding endpoints makes no difference.

5.2 Cumulative Distribution Function (CDF)

The CDF gives the total probability accumulated up to a point x:

CDF definition F(x) = P(X ≤ x) = ∫−∞x f(t) dt
Properties of the CDF:
• F is non-decreasing: if a < b then F(a) ≤ F(b)
• F(−∞) = 0 and F(∞) = 1
• P(a ≤ X ≤ b) = F(b) − F(a)
• f(x) = F′(x) — the PDF is the derivative of the CDF

5.3 Empirical CDF (F̂n)

Given observed data x₁, ..., xₙ, the empirical CDF is a step function:

Empirical CDF F̂n(x) = (number of observations ≤ x) / n
Exam application (2022): You may be given a plot of the empirical CDF and asked to read off quantiles or the IQR. To find the IQR from an empirical CDF plot:
• Find Q1: the x-value where F̂n(x) = 0.25
• Find Q3: the x-value where F̂n(x) = 0.75
• IQR = Q3 − Q1
Read these values by drawing horizontal lines at 0.25 and 0.75 on the y-axis, then dropping down to the x-axis.
Worked Example — Uniform Distribution

X ~ Uniform(0, 5). The PDF is f(x) = 1/5 for 0 ≤ x ≤ 5.

Find P(1 ≤ X ≤ 3)
P(1 ≤ X ≤ 3) = ∫13 (1/5) dx = (3 − 1)/5 = 2/5 = 0.4
CDF
F(x) = x/5 for 0 ≤ x ≤ 5
P(1 ≤ X ≤ 3) = F(3) − F(1) = 3/5 − 1/5 = 0.4 (same answer)

R Functions for Empirical CDF

ecdf(data) — creates the empirical CDF function.
plot(ecdf(data)) — plots the step-function CDF.
quantile(data, 0.25) — extracts quantiles directly.
Concept

6. Normal Distribution & Z-Scores

The normal (Gaussian) distribution is the most important continuous distribution. It is symmetric, bell-shaped, and fully described by its mean μ and standard deviation σ.

Z-Score (Standardization) Z = (X − μ) / σ

The Z-score tells you how many standard deviations X is from the mean. After standardizing, Z ~ N(0, 1).

The 68-95-99.7 Rule

±1σ: 68.27% of data
±2σ: 95.45% of data
±3σ: 99.73% of data

R Functions

pnorm(x, mean, sd) — P(X ≤ x), cumulative probability (area to the left).
qnorm(p, mean, sd) — Inverse: find x such that P(X ≤ x) = p.
dnorm(x, mean, sd) — The density (height of curve), NOT a probability.

Symmetry Property

P(Z > z) = P(Z < −z)
Because the standard normal is symmetric around 0, the area to the right of z equals the area to the left of −z. This means pnorm(−z) = 1 − pnorm(z).

Key Quantiles (appear frequently on exams)

p qnorm(p) Typical use
0.025 −1.96 95% CI lower bound
0.05 −1.645 90% CI lower bound / one-sided α=0.05
0.10 −1.28 80% CI lower bound / one-sided α=0.10
0.90 1.28 By symmetry: qnorm(0.90) = −qnorm(0.10)
0.95 1.645 By symmetry: qnorm(0.95) = −qnorm(0.05)
0.975 1.96 95% CI upper bound
Common exam mistake: Confusing pnorm and qnorm. Remember: pnorm takes an x-value and returns a probability; qnorm takes a probability and returns an x-value. They are inverses of each other.
Worked Example

What percentage of observations fall within 1.3 standard deviations of the mean?

Calculation
P(−1.3 ≤ Z ≤ 1.3) = P(Z ≤ 1.3) − P(Z ≤ −1.3)
= pnorm(1.3) − pnorm(−1.3)
= 0.9032 − 0.0968
= 0.8064 = 80.64%
Symmetry shortcut: P(−z ≤ Z ≤ z) = 2 · pnorm(z) − 1. So for z = 1.3: 2 · 0.9032 − 1 = 0.8064.
Concept

7. Central Limit Theorem (CLT)

The CLT is arguably the most important theorem in statistics. It says that the sampling distribution of the sample mean approaches a normal distribution as n increases — regardless of the population's shape.

CLT Statement X̄n ~ N(μ, σ / √n)    approximately, for large n

Conditions

  • Observations are independent (random sampling or n < 10% of population)
  • n ≥ 30 (rule of thumb), OR the population itself is normal

Variance of the Sample Mean

Derivation (appears on exams!) Var(X̄n) = Var(1/n · ∑Xi) = 1/n2 · ∑Var(Xi) = 1/n2 · nσ2 = σ2/n
Critical step: The sum of variances only works because the Xi are independent. If they're dependent, you need covariance terms: Var(∑Xi) = ∑Var(Xi) + 2∑i<jCov(Xi,Xj).
Standard Error (SE): SE = σ / √n. This is the standard deviation of X̄. As n increases, SE decreases — more data = more precise estimates.
Worked Example

A population has μ = 50, σ = 12. For a sample of n = 36, find P(X̄ > 53).

Step 1: Find the standard error
SE = σ / √n = 12 / √36 = 12 / 6 = 2
Step 2: Standardize
Z = (53 − 50) / 2 = 1.5
Step 3: Find probability
P(X̄ > 53) = P(Z > 1.5) = 1 − pnorm(1.5) = 1 − 0.9332 = 0.0668
Summary

8. Distribution Cheat Sheet

Discrete Distributions

Distribution Use when... E(X) SD(X) R function
Binomial # successes in n trials np √(np(1−p)) dbinom / pbinom
Geometric Trials until 1st success 1/p √((1−p)/p²) dgeom / pgeom
Neg. Binomial Trials until k-th success k/p √(k(1−p)/p²) dnbinom / pnbinom
Poisson # events in fixed interval λ √λ dpois / ppois

Continuous Distributions

Distribution Use when... E(X) Var(X) R function
Normal Continuous, symmetric data μ σ2 pnorm / qnorm / dnorm
Uniform(a,b) Equal probability over interval (a+b)/2 (b−a)2/12 punif / qunif / dunif
Decision flowchart:
Counting successes in n trials? → Binomial
Counting events in an interval? → Poisson
Waiting for first success? → Geometric
Waiting for k-th success? → Negative Binomial
Continuous & bell-shaped? → Normal
Continuous & equally likely over a range? → Uniform
PDF vs PMF reminder:
• Discrete distributions use a PMF: P(X = x) gives an actual probability.
• Continuous distributions use a PDF: f(x) is a density, not a probability. You must integrate to get P(a ≤ X ≤ b).

Exercise 1 — Binomial: Online Dating
3 pts
A 2015 Pew Research poll found that 15% of American adults have used online dating sites or apps. In a random sample of 20 American adults, what is the probability that at least one has used online dating?
dbinom(0, size = 20, prob = 0.15) pbinom(0, size = 20, prob = 0.15)
Distribution: Binomial with n = 20, p = 0.15

Strategy: "At least one" = complement of "none"
P(X ≥ 1) = 1 − P(X = 0)

Calculate P(X = 0):
P(X = 0) = C(20, 0) · 0.150 · 0.8520
= 1 · 1 · 0.8520
= 0.0388

In R: dbinom(0, 20, 0.15) = 0.03876

Final answer:
P(X ≥ 1) = 1 − 0.0388 = 0.9612 ≈ 96.1%

There is approximately a 96.1% probability that at least one person in the sample has used online dating.

2 pts
A call center receives an average of 4.5 calls per hour. a) What is P(X = 6) in a given hour? b) What is P(X ≤ 2) in a given hour?
dpois(6, lambda = 4.5) ppois(2, lambda = 4.5)
Distribution: Poisson with λ = 4.5

a) P(X = 6):
P(X = 6) = 4.56 · e−4.5 / 6!
= 8303.77 · 0.01111 / 720
= 0.1281
In R: dpois(6, 4.5) = 0.1281

b) P(X ≤ 2):
P(X ≤ 2) = P(X=0) + P(X=1) + P(X=2)
= e−4.5 + 4.5 · e−4.5 + 4.52/2 · e−4.5
= e−4.5(1 + 4.5 + 10.125)
= 0.01111 · 15.625
= 0.1736
In R: ppois(2, 4.5) = 0.1736

Exercise 3 — Expected Value: Card Game (2023 Retake)
2 pts
You draw one card from a standard deck of 52 cards. The payoff rules are: • Red card (hearts or diamonds): win nothing ($0) • Spade: win $5 • Club (not the ace of clubs): win $10 • Ace of clubs: win $40 (total — NOT $10 + $30) Find E(X), the expected winnings per draw.
Outcomes and probabilities:
• Red card: 26/52 = 1/2, payoff = $0
• Spade (all 13 spades): 13/52 = 1/4, payoff = $5
• Club (not ace): 12/52 = 3/13, payoff = $10
• Ace of clubs: 1/52, payoff = $40

Check probabilities sum to 1: 26 + 13 + 12 + 1 = 52

E(X) = ∑ xi · P(xi):
= 0 · (26/52) + 5 · (13/52) + 10 · (12/52) + 40 · (1/52)
= 0 + 65/52 + 120/52 + 40/52
= 225/52
= $4.33

Breakdown: 0 + 1.25 + 2.308 + 0.769 = $4.33

On average, you win about $4.33 per draw.

Exercise 4 — Normal: Z-Score Interval
3 pts
A variable follows a normal distribution with mean μ and standard deviation σ. What percentage of observations are within 1.3 standard deviations of the mean?
pnorm(1.3) - pnorm(-1.3) # or equivalently: 2 * pnorm(1.3) - 1
We need: P(μ − 1.3σ ≤ X ≤ μ + 1.3σ)

Standardize:
P(−1.3 ≤ Z ≤ 1.3) = P(Z ≤ 1.3) − P(Z ≤ −1.3)

Using R / Z-table:
pnorm(1.3) = 0.9032
pnorm(−1.3) = 0.0968

P(−1.3 ≤ Z ≤ 1.3) = 0.9032 − 0.0968 = 0.8064

Equivalently: 2 · pnorm(1.3) − 1 = 2(0.9032) − 1 = 0.8064

Answer: 80.64% of observations fall within 1.3 standard deviations of the mean.

Exercise 5 — CLT: Variance of the Sample Mean
3 pts
Let X₁, X₂, ..., Xₙ be n independent random variables with E(Xᵢ) = μ and Var(Xᵢ) = σ² for all i. Define X̄ₙ = (1/n) · ∑Xᵢ. Calculate Var(X̄ₙ). Show your derivation step by step.
Step 1: Write out X̄ₙ
X̄ₙ = (1/n) · (X₁ + X₂ + ... + Xₙ)

Step 2: Apply the constant-factor rule
Var(cY) = c2 · Var(Y), where c = 1/n:
Var(X̄ₙ) = (1/n)2 · Var(X₁ + X₂ + ... + Xₙ)

Step 3: Use independence to split the sum
Since the Xᵢ are independent, Var(X₁ + ... + Xₙ) = Var(X₁) + ... + Var(Xₙ)
= n · σ2

Step 4: Combine
Var(X̄ₙ) = (1/n2) · nσ2 = σ2 / n

Key insight: Independence is essential. Without it, we'd need:
Var(∑Xᵢ) = ∑Var(Xᵢ) + 2∑i<jCov(Xᵢ, Xⱼ)

Exercise 6 — Binomial with R Output
2 pts
In a random sample of 5 people, each independently has a probability of 0.342 of having a certain characteristic. What is the probability that exactly 2 of the 5 people have this characteristic?
dbinom(2, size = 5, prob = 0.342) pbinom(2, size = 5, prob = 0.342)
Distribution: Binomial with n = 5, p = 0.342, x = 2

Formula:
P(X = 2) = C(5, 2) · 0.3422 · (1 − 0.342)3
= 10 · 0.11696 · 0.6583
= 10 · 0.11696 · 0.28484
= 0.3331

In R: dbinom(2, 5, 0.342) = 0.3331

Note: pbinom(2, 5, 0.342) = 0.8060 would give P(X ≤ 2), which is the cumulative probability. The question asks for exactly 2, so we use dbinom.

Exercise 7 — Normal: Using pnorm and qnorm
3 pts
Exam scores follow a normal distribution with μ = 72 and σ = 8. a) What proportion of students scored below 60? b) What score marks the top 5% of students? c) What is the IQR of the exam scores?
pnorm(60, mean = 72, sd = 8) qnorm(0.95, mean = 72, sd = 8) qnorm(0.75, 72, 8) - qnorm(0.25, 72, 8)
a) P(X < 60):
Z = (60 − 72) / 8 = −1.5
P(X < 60) = pnorm(−1.5) = 0.0668 → about 6.7% scored below 60.

b) Top 5% cutoff:
We need x such that P(X ≤ x) = 0.95
x = qnorm(0.95, 72, 8) = 72 + 1.645 · 8 = 72 + 13.16 = 85.16
Students scoring above ~85.2 are in the top 5%.

c) IQR:
Q1 = qnorm(0.25, 72, 8) = 72 + (−0.6745) · 8 = 72 − 5.396 = 66.60
Q3 = qnorm(0.75, 72, 8) = 72 + 0.6745 · 8 = 72 + 5.396 = 77.40
IQR = 77.40 − 66.60 = 10.79

Exercise 8 — Negative Binomial
2 pts
A quality inspector checks items on a production line. Each item independently has a 10% chance of being defective. a) What is the expected number of items inspected until finding the 3rd defective item? b) What is the probability that exactly the 8th item inspected is the 3rd defective?
Parameters: k = 3 (3rd defective), p = 0.10

a) Expected number of inspections:
E(X) = k/p = 3 / 0.10 = 30 items

b) P(X = 8):
P(X = 8) = C(7, 2) · 0.13 · 0.95
= 21 · 0.001 · 0.59049
= 0.01240

Reading the formula: in the first 7 items, exactly 2 must be defective (C(7,2) ways), the 8th item is the 3rd defective, and the remaining 5 items are non-defective (0.95).

In R: dnbinom(5, size=3, prob=0.1) = 0.01240
(R counts failures: x = 8 − 3 = 5 failures)
