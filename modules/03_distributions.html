Concept

1. Binomial Distribution

The binomial distribution models the number of successes in a fixed number of independent trials, each with the same probability of success.

Conditions (all must hold)

  • Fixed number of trials n
  • Each trial has exactly two outcomes (success/failure)
  • Trials are independent
  • Probability of success p is constant across trials
Binomial PMF P(X = x) = C(n, x) · px · (1 − p)n − x
Expected Value & Standard Deviation E(X) = n · p      SD(X) = √(n · p · (1 − p))

R Functions

dbinom(x, size, prob) — P(X = x), the exact probability of x successes.
pbinom(x, size, prob) — P(X ≤ x), the cumulative probability up to x.
Worked Example

In a random sample of 25 households, 12% own an electric vehicle. What is P(X = 3)?

Here: n = 25, p = 0.12, x = 3.

Calculation
P(X = 3) = C(25, 3) · 0.123 · 0.8822
= 2300 · 0.001728 · 0.05863
= 0.2330

In R: dbinom(3, 25, 0.12) = 0.2330
Exam tip: "At least one" means P(X ≥ 1) = 1 − P(X = 0). This complement trick appears constantly.
Concept

2. Poisson Distribution

The Poisson distribution models the number of events in a fixed interval (time, area, volume) when events occur independently at a constant average rate.

Conditions

  • Events occur independently
  • The average rate λ is constant over the interval
  • Two events cannot occur at exactly the same instant
Poisson PMF P(X = x) = λx · e−λ / x!
Expected Value & Standard Deviation E(X) = λ      Var(X) = λ      SD(X) = √λ

R Functions

dpois(x, lambda) — P(X = x)
ppois(x, lambda) — P(X ≤ x)
Worked Example

A hospital emergency room receives an average of 6 patients per hour. What is P(X = 4)?

Calculation
P(X = 4) = 64 · e−6 / 4!
= 1296 · 0.002479 / 24
= 0.1339

In R: dpois(4, 6) = 0.1339
Poisson as Binomial approximation: When n is large and p is small, Bin(n, p) ≈ Poisson(λ = np). Rule of thumb: n ≥ 100, p ≤ 0.01.
Concept

3. Geometric Distribution

The geometric distribution models the number of trials until the first success. Each trial is independent with constant success probability p.

Geometric PMF P(X = x) = (1 − p)x − 1 · p     for x = 1, 2, 3, ...
Expected Value & Variance E(X) = 1/p      Var(X) = (1 − p) / p2
Worked Example

A free-throw shooter has a 70% success rate. What is the probability it takes exactly 3 attempts to make the first basket?

Calculation
P(X = 3) = (1 − 0.7)2 · 0.7 = 0.32 · 0.7 = 0.09 · 0.7 = 0.063

In R: dgeom(2, 0.7) = 0.063
Note: R's dgeom counts failures (x−1), not trials.
Watch out: The geometric distribution is memoryless. P(X > s + t | X > s) = P(X > t). Past failures don't affect future probability.
Concept

4. Negative Binomial Distribution

The negative binomial generalizes the geometric: it models the number of trials until the k-th success.

Negative Binomial PMF P(X = x) = C(x − 1, k − 1) · pk · (1 − p)x − k     for x = k, k+1, k+2, ...
Expected Value & Variance E(X) = k/p      Var(X) = k(1 − p) / p2
Relationship: Geometric is just Negative Binomial with k = 1.
Worked Example

A recruiter interviews candidates, and each has a 20% chance of being hired. How many interviews until 3 hires, on average?

Calculation
E(X) = k/p = 3 / 0.20 = 15 interviews
Concept

5. Normal Distribution & Z-Scores

The normal (Gaussian) distribution is the most important continuous distribution. It is symmetric, bell-shaped, and fully described by its mean μ and standard deviation σ.

Z-Score (Standardization) Z = (X − μ) / σ

The Z-score tells you how many standard deviations X is from the mean. After standardizing, Z ~ N(0, 1).

The 68-95-99.7 Rule

±1σ: 68.27% of data
±2σ: 95.45% of data
±3σ: 99.73% of data

R Functions

pnorm(x, mean, sd) — P(X ≤ x), cumulative probability (area to the left).
qnorm(p, mean, sd) — Inverse: find x such that P(X ≤ x) = p.
dnorm(x, mean, sd) — The density (height of curve), NOT a probability.
Worked Example

What percentage of observations fall within 1.3 standard deviations of the mean?

Calculation
P(−1.3 ≤ Z ≤ 1.3) = P(Z ≤ 1.3) − P(Z ≤ −1.3)
= pnorm(1.3) − pnorm(−1.3)
= 0.9032 − 0.0968
= 0.8064 = 80.64%
Symmetry shortcut: P(−z ≤ Z ≤ z) = 2 · pnorm(z) − 1. So for z = 1.3: 2 · 0.9032 − 1 = 0.8064.
Concept

6. Central Limit Theorem (CLT)

The CLT is arguably the most important theorem in statistics. It says that the sampling distribution of the sample mean approaches a normal distribution as n increases — regardless of the population's shape.

CLT Statement X̄n ~ N(μ, σ / √n)    approximately, for large n

Conditions

  • Observations are independent (random sampling or n < 10% of population)
  • n ≥ 30 (rule of thumb), OR the population itself is normal

Variance of the Sample Mean

Derivation (appears on exams!) Var(X̄n) = Var(1/n · ∑Xi) = 1/n2 · ∑Var(Xi) = 1/n2 · nσ2 = σ2/n
Critical step: The sum of variances only works because the Xi are independent. If they're dependent, you need covariance terms: Var(∑Xi) = ∑Var(Xi) + 2∑i<jCov(Xi,Xj).
Standard Error (SE): SE = σ / √n. This is the standard deviation of X̄. As n increases, SE decreases — more data = more precise estimates.
Worked Example

A population has μ = 50, σ = 12. For a sample of n = 36, find P(X̄ > 53).

Step 1: Find the standard error
SE = σ / √n = 12 / √36 = 12 / 6 = 2
Step 2: Standardize
Z = (53 − 50) / 2 = 1.5
Step 3: Find probability
P(X̄ > 53) = P(Z > 1.5) = 1 − pnorm(1.5) = 1 − 0.9332 = 0.0668
Summary

7. Distribution Cheat Sheet

Distribution Use when... E(X) Var(X) R function
Binomial # successes in n trials np np(1−p) dbinom / pbinom
Poisson # events in fixed interval λ λ dpois / ppois
Geometric Trials until 1st success 1/p (1−p)/p2 dgeom / pgeom
Neg. Binomial Trials until k-th success k/p k(1−p)/p2 dnbinom / pnbinom
Normal Continuous, symmetric data μ σ2 pnorm / qnorm
Decision flowchart:
Counting successes in n trials? → Binomial
Counting events in an interval? → Poisson
Waiting for first success? → Geometric
Waiting for k-th success? → Negative Binomial
Continuous & bell-shaped? → Normal

Exercise 1 — Binomial: Online Dating
3 pts
A 2015 Pew Research poll found that 15% of American adults have used online dating sites or apps. In a random sample of 20 American adults, what is the probability that at least one has used online dating?
dbinom(0, size = 20, prob = 0.15) pbinom(0, size = 20, prob = 0.15)
Distribution: Binomial with n = 20, p = 0.15

Strategy: "At least one" = complement of "none"
P(X ≥ 1) = 1 − P(X = 0)

Calculate P(X = 0):
P(X = 0) = C(20, 0) · 0.150 · 0.8520
= 1 · 1 · 0.8520
= 0.0388

In R: dbinom(0, 20, 0.15) = 0.03876

Final answer:
P(X ≥ 1) = 1 − 0.0388 = 0.9612 ≈ 96.1%

There is approximately a 96.1% probability that at least one person in the sample has used online dating.

Exercise 2 — Poisson: Call Center
2 pts
A call center receives an average of 4.5 calls per hour. a) What is P(X = 6) in a given hour? b) What is P(X ≤ 2) in a given hour?
dpois(6, lambda = 4.5) ppois(2, lambda = 4.5)
Distribution: Poisson with λ = 4.5

a) P(X = 6):
P(X = 6) = 4.56 · e−4.5 / 6!
= 8303.77 · 0.01111 / 720
= 0.1281
In R: dpois(6, 4.5) = 0.1281

b) P(X ≤ 2):
P(X ≤ 2) = P(X=0) + P(X=1) + P(X=2)
= e−4.5 + 4.5 · e−4.5 + 4.52/2 · e−4.5
= e−4.5(1 + 4.5 + 10.125)
= 0.01111 · 15.625
= 0.1736
In R: ppois(2, 4.5) = 0.1736

Exercise 3 — Expected Value: Card Game
2 pts
You draw one card from a standard deck of 52 cards. The payoff rules are: • Red card (hearts or diamonds): win nothing (0) • Spade (not ace): win 5 • Club (not ace): win 10 • Ace of clubs: win 40 Find E(X), the expected winnings per draw.
Outcomes and probabilities:
• Red card: 26/52 = 1/2, payoff = 0
• Spade (13 cards, including ace of spades): 13/52 = 1/4, payoff = 5
• Club (not ace): 12/52 = 3/13, payoff = 10
• Ace of clubs: 1/52, payoff = 40

E(X) = ∑ xi · P(xi):
= 0 · (26/52) + 5 · (13/52) + 10 · (12/52) + 40 · (1/52)
= 0 + 65/52 + 120/52 + 40/52
= 225/52
= 4.327 ≈ 4.33

On average, you win about €4.33 per draw.

Exercise 4 — Normal: Z-Score Interval
3 pts
A variable follows a normal distribution with mean μ and standard deviation σ. What percentage of observations are within 1.3 standard deviations of the mean?
pnorm(1.3) - pnorm(-1.3) # or equivalently: 2 * pnorm(1.3) - 1
We need: P(μ − 1.3σ ≤ X ≤ μ + 1.3σ)

Standardize:
P(−1.3 ≤ Z ≤ 1.3) = P(Z ≤ 1.3) − P(Z ≤ −1.3)

Using R / Z-table:
pnorm(1.3) = 0.9032
pnorm(−1.3) = 0.0968

P(−1.3 ≤ Z ≤ 1.3) = 0.9032 − 0.0968 = 0.8064

Equivalently: 2 · pnorm(1.3) − 1 = 2(0.9032) − 1 = 0.8064

Answer: 80.64% of observations fall within 1.3 standard deviations of the mean.

Exercise 5 — CLT: Variance of the Sample Mean
3 pts
Let X₁, X₂, ..., Xₙ be n independent random variables with E(Xᵢ) = μ and Var(Xᵢ) = σ² for all i. Define X̄ₙ = (1/n) · ∑Xᵢ. Calculate Var(X̄ₙ). Show your derivation step by step.
Step 1: Write out X̄ₙ
X̄ₙ = (1/n) · (X₁ + X₂ + ... + Xₙ)

Step 2: Apply the constant-factor rule
Var(cY) = c2 · Var(Y), where c = 1/n:
Var(X̄ₙ) = (1/n)2 · Var(X₁ + X₂ + ... + Xₙ)

Step 3: Use independence to split the sum
Since the Xᵢ are independent, Var(X₁ + ... + Xₙ) = Var(X₁) + ... + Var(Xₙ)
= n · σ2

Step 4: Combine
Var(X̄ₙ) = (1/n2) · nσ2 = σ2 / n

Key insight: Independence is essential. Without it, we'd need:
Var(∑Xᵢ) = ∑Var(Xᵢ) + 2∑i<jCov(Xᵢ, Xⱼ)

Exercise 6 — Binomial with R Output
2 pts
In a random sample of 5 people, each independently has a probability of 0.342 of having a certain characteristic. What is the probability that exactly 2 of the 5 people have this characteristic?
dbinom(2, size = 5, prob = 0.342) pbinom(2, size = 5, prob = 0.342)
Distribution: Binomial with n = 5, p = 0.342, x = 2

Formula:
P(X = 2) = C(5, 2) · 0.3422 · (1 − 0.342)3
= 10 · 0.11696 · 0.6583
= 10 · 0.11696 · 0.28484
= 0.3331

In R: dbinom(2, 5, 0.342) = 0.3331

Note: pbinom(2, 5, 0.342) = 0.8060 would give P(X ≤ 2), which is the cumulative probability. The question asks for exactly 2, so we use dbinom.
