Module 6: Linear Regression

Interpret slopes, R², AIC backward elimination — every exam
Week 1 • Day 6
Concept

1. Simple Linear Regression

We fit a straight line through data to predict a response variable y from one predictor x.

Simple Linear Regression Model
ŷ = b₀ + b₁x
  • b₀ (intercept): predicted value of y when x = 0
  • b₁ (slope): for each 1-unit increase in x, y changes by b₁ on average
Residual
Residual = y − ŷ = observed − predicted

A positive residual means the model underestimated; a negative residual means it overestimated.

Slope from correlation
b₁ = r · (sy / sx)
Reading R output: In the Coefficients table, the Estimate column gives b₀ (Intercept row) and b₁ (predictor row). The Std. Error column gives SEb. The t value column is Estimate / Std. Error.
Interpretation template: "For each additional [unit of x], we expect [y] to [increase/decrease] by [|b₁|] [units of y], on average."
Concept

2. Multiple Linear Regression

With multiple predictors, the model extends to:

Multiple Linear Regression
ŷ = b₀ + b₁x₁ + b₂x₂ + ... + bkxk
"All else being equal" interpretation:
Each slope bj tells you the effect of xj on y while holding all other predictors constant.

Template: "All else being equal, for each 1-unit increase in xj, y changes by bj on average."

Categorical Predictors (Indicator Variables)

R automatically creates indicator (dummy) variables for categorical predictors. For example, if Sex has levels Female and Male:

  • SexM = 1 if the observation is male, 0 if female
  • Female is the reference category (absorbed into the intercept)
Example with indicator variable:
ŷ = 12.5 + 3.2 · Weight − 1.8 · SexM

For females (SexM = 0): ŷ = 12.5 + 3.2 · Weight
For males (SexM = 1): ŷ = (12.5 − 1.8) + 3.2 · Weight = 10.7 + 3.2 · Weight

Interpretation of SexM coefficient: "All else being equal, males have a predicted y that is 1.8 units lower than females."

Interaction Terms

An interaction term like Bwt:SexM means the slope of Bwt differs between males and females.

Model with interaction
ŷ = b₀ + b₁·Bwt + b₂·SexM + b₃·Bwt:SexM

The interaction coefficient b₃ tells you how much steeper (or flatter) the slope is for the non-reference category.

Concept

3. R² and Adjusted R²

R² measures the proportion of variance in y explained by the model. Ranges from 0 to 1.

R² (Coefficient of Determination)
R² = 1 − Σ(yi − ŷi)² / Σ(yi − ȳ)² = 1 − SSRes / SSTot
For simple regression: R² = r² (the square of the correlation coefficient).

Problem: R² always increases when you add more predictors, even useless ones. Solution:

Adjusted R²
R²adj = 1 − [Σ(yi − ŷi)² / (n − k − 1)] / [Σ(yi − ŷi)² / (n − 1)]

Equivalently:

Adjusted R² from residual standard error
R²adj = 1 − (Residual SE)² / Var(y)
Computing R²adj from R output:
1. Find Residual standard error in the output (e.g., 15.72)
2. Compute Var(y) separately (e.g., from var(data$y))
3. R²adj = 1 − (15.72)² / Var(y)
R² Always increases with more predictors.
Use for simple regression or reporting.
R²adj Penalizes for extra predictors.
Use for comparing models with different numbers of predictors.
Concept

4. Hypothesis Testing for Coefficients

For each predictor, R tests whether the coefficient is significantly different from zero:

Hypotheses
H₀: βj = 0   (predictor has no linear relationship with y)
HA: βj ≠ 0   (predictor does have a linear relationship with y)
Test statistic
T = (bj − 0) / SEbj     with df = n − k − 1

where k = number of predictors in the model.

Confidence Interval for a coefficient
bj ± t* · SEbj
Decision rules (equivalent):
• If p-value < α, reject H₀ → predictor is significant
• If the CI for βj does NOT contain 0, the predictor is significant
• If |T| > t*crit, reject H₀
Exam trap: The degrees of freedom for the t-test is df = n − k − 1 (residual df), NOT n − 1. Read the residual df from the R output line: Residual standard error: XX on YY degrees of freedom. YY is your df.
Concept

5. AIC Backward Elimination

AIC (Akaike Information Criterion) balances model fit against complexity:

AIC (simplified form for regression)
AIC = constant + n · ln(RSS / n) + 2k

Lower AIC = better model. The step() function in R performs backward elimination automatically.

How backward elimination works:

  1. Start with the full model (all predictors). Note the current AIC.
  2. Try removing each variable one at a time. For each removal, compute the new AIC.
  3. Remove the variable whose removal gives the LOWEST AIC.
  4. Repeat until removing any remaining variable would increase AIC.
  5. The <none> row shows the AIC of keeping all remaining variables.
EXAM TRAP: In the step() output, the rows are often permuted/shuffled (not in the order you might expect). You must scan ALL rows to find the one with the minimum AIC value. Do NOT just pick the first or last row!
Reading step() output:
RemoveDfAIC
<none>4415.11
− age14413.35
− height14418.22
− parity14414.80

<none> AIC = 4415.11 (keeping everything).
Removing age gives AIC = 4413.35, which is lower than 4415.11 → removing age improves the model.
4413.35 is the minimum AIC → remove age.
AIC and R²adj relationship:
Lower AIC generally corresponds to higher R²adj. A model with lower AIC will typically have a larger (or equal) adjusted R² compared to a model with higher AIC.
Concept

6. Residual Analysis — Checking Model Assumptions

After fitting a linear model, R produces 4 diagnostic plots via plot(model). Each checks a specific assumption:

Plot 1: Residuals vs Fitted

  • Checks: Linearity and constant variance
  • Good: Random scatter around 0, no pattern (no curve, no fan shape)
  • Bad: U-shape or curve → non-linear relationship. Fan/funnel shape → non-constant variance

Plot 2: Normal Q-Q

  • Checks: Normality of residuals
  • Good: Points fall approximately on the diagonal line
  • Bad: Systematic deviations from the line (heavy tails, skewness)

Plot 3: Scale-Location (Spread-Location)

  • Checks: Homoscedasticity (constant variance)
  • Good: Horizontal red line, points evenly spread
  • Bad: Increasing/decreasing trend → variance changes with fitted values

Plot 4: Residuals vs Leverage

  • Checks: Influential observations
  • Good: All points inside Cook's distance contours (dashed lines)
  • Bad: Points beyond Cook's distance > 0.5 (or > 1) are highly influential
Quick memory trick for the exam:
1. Residuals vs Fitted → Linearity + Equal variance
2. Normal Q-Q → Normality
3. Scale-Location → Spread (homoscedasticity)
4. Residuals vs Leverage → Influential points
Cheat Sheet

7. Interaction Terms & Module Summary

Writing out the simplified model for a specific category

When a model has indicator variables and interactions, substitute the category value to simplify:

Example: Cat heart weight model
Full model: ŷ = 2.98 + 2.64·Bwt − 4.17·SexM + 1.68·Bwt:SexM

For female cats (SexM = 0):
ŷ = 2.98 + 2.64·Bwt − 4.17(0) + 1.68·Bwt(0)
ŷ = 2.98 + 2.64·Bwt

For male cats (SexM = 1):
ŷ = 2.98 + 2.64·Bwt − 4.17(1) + 1.68·Bwt(1)
ŷ = (2.98 − 4.17) + (2.64 + 1.68)·Bwt
ŷ = −1.19 + 4.32·Bwt

Module 6 Summary

  1. Simple regression: ŷ = b₀ + b₁x, slope = r · (sy/sx)
  2. Residual = observed − predicted = y − ŷ
  3. Multiple regression: "all else being equal" interpretation for each slope
  4. Indicator variables: reference category absorbed into intercept
  5. Interaction terms: different slopes for different categories
  6. R² = 1 − SSRes/SSTot; for simple regression R² = r²
  7. R²adj = 1 − (Residual SE)²/Var(y) — penalizes extra predictors
  8. T-test for coefficients: T = bj/SE, df = n − k − 1
  9. AIC backward elimination: remove variable giving lowest AIC; stop when <none> is lowest
  10. 4 diagnostic plots: linearity, normality, homoscedasticity, influential points

3 pts • Exam classic
A simple linear regression predicts the calorie content of Starbucks food items from their carbohydrate content (in grams). The R output gives:
Coefficients: Estimate Std. Error t value Pr(>|t|) (Intercept) 146.0200 25.9186 5.634 2.93e-07 Carb 4.2971 0.5424 7.925 1.65e-11 Residual standard error: 78.26 on 74 degrees of freedom Multiple R-squared: 0.4590 Adjusted R-squared: 0.4517
a) Write the equation of the regression model. b) Interpret the slope of carbohydrate in context. c) A food item has 350 calories and 45g carbs. Compute the residual. Did the model overestimate or underestimate?
a) Equation
b) Interpretation
c) Residual
a) ŷ = 146.02 + 4.2971 · Carb

b) For each additional gram of carbohydrate, the predicted calorie content of Starbucks food items increases by approximately 4.30 calories, on average.

c)
Predicted: ŷ = 146.02 + 4.2971 · 45 = 146.02 + 193.37 = 339.39 calories
Residual = y − ŷ = 350 − 339.39 = +10.61 calories

The residual is positive, so the model underestimated the calories for this food item by about 10.6 calories.

b) Interpretations

Exercise 4 — Interaction Term
3 pts • From 2023 exam
A model predicts cat heart weight (Hwt, in grams) from body weight (Bwt, in kg) and sex. The coefficient table:
Coefficients: Estimate Std. Error t value Pr(>|t|) (Intercept) 2.9813 1.8428 1.618 0.1079 Bwt 2.6364 0.6837 3.857 0.0002 SexM -4.1654 2.5635 -1.625 0.1064 Bwt:SexM 1.6763 1.0120 1.657 0.0997
a) Write the estimated model for a male cat. Simplify the equation. b) Interpret the coefficient of SexM in context. c) Is the interaction term Bwt:SexM significant at the α = 0.05 level? Hint: pt(c(1.656, 1.978), df=140, lower.tail=FALSE) gives 0.0500 and 0.0249.
a) Model for male cats
b) Interpret SexM
c) Significance of interaction
a) Full model: ŷ = 2.98 + 2.64·Bwt − 4.17·SexM + 1.68·Bwt:SexM

For male cats, SexM = 1:
ŷ = 2.98 + 2.64·Bwt − 4.17(1) + 1.68·Bwt(1)
ŷ = (2.98 − 4.17) + (2.64 + 1.68)·Bwt
ŷ = −1.19 + 4.32·Bwt

b) The coefficient of SexM (−4.17) represents the difference in predicted heart weight between male and female cats when Bwt = 0. However, because the model includes an interaction term (Bwt:SexM), this coefficient alone does NOT represent the overall difference between sexes — the difference depends on body weight. At Bwt = 0, male cats have a predicted heart weight 4.17 grams lower than female cats.

c) The t-value for Bwt:SexM is 1.657.
From the hint: pt(1.656, df=140, lower.tail=FALSE) = 0.0500
Since this is a two-sided test, p-value = 2 × 0.0500 = 0.1000.

Actually, looking more carefully: t = 1.657 > 1.656, so the one-sided p-value is slightly less than 0.0500.
But the two-sided p-value = 2 × (slightly less than 0.05) = slightly less than 0.10, which matches the output p = 0.0997.

Since p = 0.0997 > 0.05, the interaction term is NOT significant at the α = 0.05 level. There is insufficient evidence to conclude that the relationship between body weight and heart weight differs between male and female cats.

Exercise 5 — Adjusted R² Comparison
2 pts • From 2023 retake
After backward elimination, model_step has AIC = 1409.6 while model_full has AIC = 1416.9. Is the adjusted R² of model_step smaller or larger than that of model_full? Justify your answer using the relationship between AIC and adjusted R².
R²adj of model_step is larger than that of model_full.

Justification:
Both AIC and R²adj penalize for the number of parameters in the model.

• AIC = constant + n·ln(RSS/n) + 2k — lower is better
• R²adj = 1 − [RSS/(n−k−1)] / [SSTot/(n−1)] — higher is better

When a variable is removed and AIC decreases, it means the penalty reduction (from fewer parameters) outweighs the loss in fit. The same logic applies to R²adj: removing a weak predictor increases R²adj because the penalty term shrinks more than the fit worsens.

Since model_step has lower AIC (1409.6 < 1416.9), it also has higher R²adj.

Exercise 6 — Residual Analysis
2 pts
After fitting a linear model, R produces 4 diagnostic plots. For each plot, describe: (i) what assumption it checks, and (ii) what a "good" result looks like.
Plot 1: Residuals vs Fitted
Plot 2: Normal Q-Q
Plot 3: Scale-Location
Plot 4: Residuals vs Leverage
Plot 1: Residuals vs Fitted
Checks: Linearity and constant variance (homoscedasticity).
Good: Random scatter of points around the horizontal line at 0. No curve, no fan/funnel shape.

Plot 2: Normal Q-Q
Checks: Normality of residuals.
Good: Points fall approximately along the diagonal reference line. Minor deviations at the tails are acceptable.

Plot 3: Scale-Location
Checks: Homoscedasticity (constant variance across fitted values).
Good: The red smoothing line is approximately horizontal, and points are evenly spread (no increasing or decreasing trend in spread).

Plot 4: Residuals vs Leverage
Checks: Influential observations (outliers with high leverage).
Good: No points outside Cook's distance contours (dashed lines). Cook's distance should be < 0.5 for all observations. Points with high leverage AND large residuals are problematic.
