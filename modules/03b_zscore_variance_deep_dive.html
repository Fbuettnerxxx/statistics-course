Part 1 — Z-Score
Concept

1. Was ist ein Z-Score? (Intuition)

Das Problem: Zahlen alleine sagen nichts

Stell dir vor, du schreibst eine Klausur und bekommst 75 Punkte. Gut oder schlecht?

Tja — das kommt drauf an!

Szenario A Durchschnitt: 60 Punkte
Standardabweichung: 10

Du bist 15 Punkte über dem Durchschnitt.
Das sind 1.5 Standardabweichungen drüber!
→ Du bist richtig gut.
Szenario B Durchschnitt: 80 Punkte
Standardabweichung: 10

Du bist 5 Punkte unter dem Durchschnitt.
Das sind 0.5 Standardabweichungen drunter.
→ Du bist unterdurchschnittlich.

Gleiche Punktzahl, komplett andere Bedeutung! Der Z-Score löst dieses Problem.

Die Idee in einem Satz

Z-Score = “Wie viele Standardabweichungen bin ich vom Mittelwert entfernt?”
Z-Score Formel Z = (X − μ) / σ

Dabei ist:

  • X = dein Wert (z.B. deine Punktzahl)
  • μ (mu) = der Mittelwert (Durchschnitt aller)
  • σ (sigma) = die Standardabweichung (wie stark streuen die Werte?)

Was sagt das Vorzeichen?

  • Z > 0 → Du bist über dem Durchschnitt
  • Z < 0 → Du bist unter dem Durchschnitt
  • Z = 0 → Du bist genau Durchschnitt
  • Z = 2 → Du bist 2 Standardabweichungen drüber (das ist schon sehr weit oben!)
Check dein Verständnis

Du wiegst 80kg. Der Durchschnitt ist 75kg, die Standardabweichung ist 10kg. Was ist dein Z-Score?

Z = (80 − 75) / 10 = 5 / 10 = 0.5
Du bist 0.5 Standardabweichungen über dem Durchschnitt. Leicht überdurchschnittlich.
Worked Examples

2. Z-Score berechnen — Schritt für Schritt

Wir rechnen jetzt 5 Beispiele komplett durch. Bei jedem Beispiel immer das gleiche Schema:

Kochrezept: (1) Werte rausschreiben: X = ?, μ = ?, σ = ? → (2) Einsetzen in Z = (X − μ) / σ → (3) Ausrechnen → (4) Interpretieren
Beispiel 1: Klausurnote (einfach)

Du schreibst 82 Punkte. Der Kurs hat μ = 70, σ = 8.

Schritt 1: Werte aufschreiben
X = 82, μ = 70, σ = 8
Schritt 2: Einsetzen
Z = (82 − 70) / 8
Schritt 3: Ausrechnen
Z = 12 / 8 = 1.5
Schritt 4: Interpretieren
Du bist 1.5 Standardabweichungen über dem Durchschnitt. Sehr gut!
Dein Turn

Gleicher Kurs (μ = 70, σ = 8). Dein Freund schreibt 62 Punkte. Was ist sein Z-Score?

Z = (62 − 70) / 8 = −8 / 8 = −1.0
Genau 1 Standardabweichung unter dem Durchschnitt.
Beispiel 2: Körpergröße (negativer Z-Score)

Eine Person ist 162cm groß. Die Durchschnittsgröße ist μ = 175cm, σ = 7cm.

Rechnung
X = 162, μ = 175, σ = 7
Z = (162 − 175) / 7 = −13 / 7 = −1.857

Interpretation: Fast 1.9 Standardabweichungen unter dem Durchschnitt. Deutlich kleiner als die meisten.

Beispiel 3: Temperatur (Dezimalzahlen)

Die Durchschnittstemperatur im Juli ist μ = 23.4°C mit σ = 3.2°C. Heute sind es 28.0°C.

Rechnung
Z = (28.0 − 23.4) / 3.2 = 4.6 / 3.2 = 1.4375

Interpretation: Etwa 1.44 SD wärmer als normal. Ein heißer Tag, aber keine Hitzewelle.

Beispiel 4: Gehalt (großer Z-Score)

Ein Softwareentwickler verdient 120.000€. Im Durchschnitt verdienen Leute μ = 45.000€ mit σ = 15.000€.

Rechnung
Z = (120.000 − 45.000) / 15.000 = 75.000 / 15.000 = 5.0

Interpretation: 5 Standardabweichungen über dem Durchschnitt! Das ist extrem selten — bei einer Normalverteilung wären weniger als 0.00003% der Leute so weit oben.

Beispiel 5: Von Z zurück zu X (Umkehrung!)

Manchmal kennt man den Z-Score und will den echten Wert berechnen. Dafür löst man die Formel nach X auf:

Umkehrformel X = μ + Z · σ

Beispiel: IQ hat μ = 100, σ = 15. Jemand hat Z = −2. Welcher IQ?

Rechnung
X = 100 + (−2) · 15 = 100 − 30 = 70

Ein IQ von 70 liegt 2 Standardabweichungen unter dem Durchschnitt.

Dein Turn

Körpergröße: μ = 175cm, σ = 7cm. Jemand hat Z = 1.5. Wie groß ist die Person?

X = 175 + 1.5 · 7 = 175 + 10.5 = 185.5 cm
Concept

3. Z-Score und die Normalverteilung

Die magische Transformation

Wenn X normalverteilt ist mit X ~ N(μ, σ²), dann ist der Z-Score standard-normalverteilt:

Standardisierung Wenn X ~ N(μ, σ²)   ⇒   Z = (X − μ) / σ ~ N(0, 1)

Das heißt: Egal welches μ und σ deine Daten haben — nach der Z-Transformation hat die Verteilung immer Mittelwert 0 und Standardabweichung 1. Das ist super praktisch, weil du nur eine Tabelle brauchst!

Die 68-95-99.7 Regel (Empirische Regel)

Bei einer Normalverteilung liegen:

68% der Werte zwischen Z = −1 und Z = +1
95% der Werte zwischen Z = −2 und Z = +2
99.7% der Werte zwischen Z = −3 und Z = +3

Visuell kannst du dir das so vorstellen:

Die Normalverteilungsglocke
          ___
        /     \          ← 68% hier drin
       /       \
      /    |    \        ← 95% hier drin
     /     |     \
    /      |      \      ← 99.7% hier drin
___/       |       \___
   -3  -2  -1   0  +1  +2  +3
                μ

Nur 0.3% der Werte liegen außerhalb von ±3 Standardabweichungen!

pnorm(z) — “Wie viel Prozent liegen UNTER z?”

Die Funktion pnorm(z) gibt dir die Fläche links von z. Das heißt: der Anteil aller Werte, die kleiner oder gleich z sind.

Beispiele für pnorm
  • pnorm(0) = 0.5 — 50% liegen unter dem Mittelwert (logisch!)
  • pnorm(1) = 0.8413 — 84.13% liegen unter Z = 1
  • pnorm(-1) = 0.1587 — 15.87% liegen unter Z = −1
  • pnorm(1.96) = 0.975 — 97.5% liegen unter Z = 1.96
  • pnorm(2) = 0.9772 — 97.72% liegen unter Z = 2

qnorm(p) — “Welcher Z-Wert hat p% darunter?”

qnorm ist die Umkehrfunktion von pnorm. Du gibst eine Wahrscheinlichkeit rein und bekommst den Z-Wert raus.

Beispiele für qnorm
  • qnorm(0.5) = 0 — der Median liegt bei Z = 0
  • qnorm(0.975) = 1.96 — 97.5% liegen unter Z = 1.96
  • qnorm(0.05) = −1.645 — nur 5% liegen unter Z = −1.645
  • qnorm(0.025) = −1.96 — nur 2.5% liegen unter Z = −1.96
Verwechslungsgefahr!
pnorm: du gibst einen Z-Wert rein → bekommst eine Wahrscheinlichkeit raus
qnorm: du gibst eine Wahrscheinlichkeit rein → bekommst einen Z-Wert raus
Sie sind Umkehrfunktionen: qnorm(pnorm(z)) = z
Check dein Verständnis

Wie viel Prozent der Werte liegen bei einer Normalverteilung oberhalb von Z = 1?

pnorm(1) = 0.8413 → 84.13% liegen darunter.
Also liegen 1 − 0.8413 = 0.1587 = 15.87% darüber.
Worked Examples

4. Typische Klausuraufgaben mit Z-Score

In der Klausur kommen immer wieder die gleichen Aufgabentypen. Hier sind die drei häufigsten:

Aufgabentyp 1: “Wie viel % innerhalb von k Standardabweichungen?”

Frage: Wie viel Prozent der Werte liegen innerhalb von 1.3 Standardabweichungen vom Mittelwert?

Schritt 1: Was bedeutet “innerhalb”?
Innerhalb von 1.3 SD heißt: von Z = −1.3 bis Z = +1.3
Wir suchen also: P(−1.3 < Z < 1.3)
Schritt 2: In pnorm übersetzen
P(−1.3 < Z < 1.3) = P(Z < 1.3) − P(Z < −1.3)
= pnorm(1.3) − pnorm(−1.3)
Schritt 3: Nachschlagen / berechnen
pnorm(1.3) = 0.9032
pnorm(−1.3) = 0.0968
Schritt 4: Subtrahieren
P = 0.9032 − 0.0968 = 0.8064 = 80.64%
Symmetrie-Shortcut: P(−a < Z < a) = 2 · pnorm(a) − 1
Hier: 2 · 0.9032 − 1 = 0.8064. Gleiches Ergebnis, schneller.
Aufgabentyp 2: “Cutoff für die schnellsten/besten x%”

Frage: Die Laufzeiten sind normalverteilt mit μ = 30min, σ = 4min. Welche Zeit schaffen nur die schnellsten 5%?

Schritt 1: Übersetzen
“Schnellste 5%” = die mit den niedrigsten Zeiten.
Wir suchen X so, dass P(X < x) = 0.05, also dass 5% darunter liegen.
Schritt 2: Z-Wert finden mit qnorm
Z = qnorm(0.05) = −1.645
Schritt 3: Zurück in X umrechnen
X = μ + Z · σ = 30 + (−1.645) · 4 = 30 − 6.58 = 23.42 min

Nur 5% der Läufer sind schneller als 23.42 Minuten.

Aufgabentyp 3: “P(X > k) für gegebenes X ~ N(μ, σ²)”

Frage: X ~ N(2.5, 0.4²). Berechne P(X > 3.2).

Schritt 1: Z-Score berechnen
Z = (X − μ) / σ = (3.2 − 2.5) / 0.4 = 0.7 / 0.4 = 1.75
Schritt 2: pnorm nachschlagen
P(Z < 1.75) = pnorm(1.75) = 0.9599
Schritt 3: Gegenwahrscheinlichkeit (wir wollen “größer als”!)
P(X > 3.2) = P(Z > 1.75) = 1 − pnorm(1.75) = 1 − 0.9599 = 0.0401 ≈ 4%

Key Quantiles (Tabelle für die Klausur!)

Diese Werte kommen ständig vor. Am besten auswendig lernen:

p qnorm(p) = zp Typische Verwendung
0.025 −1.96 95% CI lower bound
0.05 −1.645 90% CI / one-sided α = 0.05
0.10 −1.28 80% CI / one-sided α = 0.10
0.90 1.28 Symmetrie: qnorm(0.90) = −qnorm(0.10)
0.95 1.645 Symmetrie: qnorm(0.95) = −qnorm(0.05)
0.975 1.96 95% CI upper bound
Check dein Verständnis

Körpergröße ~ N(175, 7²). Wie groß muss man sein, um zu den größten 2.5% zu gehören?

Top 2.5% → P(X < x) = 0.975 → Z = qnorm(0.975) = 1.96
X = 175 + 1.96 · 7 = 175 + 13.72 = 188.72 cm
Klausur-Tricks

5. Z-Score Symmetrie-Tricks für die Klausur

Die Normalverteilung ist symmetrisch um 0. Das gibt dir mächtige Shortcuts:

Trick 1: Rechte Fläche = linke Fläche gespiegelt

Symmetrie-Regel P(Z > z) = P(Z < −z) = 1 − pnorm(z)
Beispiel
P(Z > 1.5) = P(Z < −1.5) = 1 − pnorm(1.5) = 1 − 0.9332 = 0.0668
Alle drei Wege geben das gleiche Ergebnis!

Trick 2: Symmetrisches Intervall

Intervall um 0 P(−a < Z < a) = 2 · pnorm(a) − 1
Beispiel
P(−1.96 < Z < 1.96) = 2 · pnorm(1.96) − 1 = 2 · 0.975 − 1 = 0.95
Deswegen ist 1.96 so wichtig — es definiert das 95%-Intervall!

Trick 3: Complement Rule

Gegenwahrscheinlichkeit P(Z > a) = 1 − P(Z < a) = 1 − pnorm(a)
ACHTUNG — DIE #1 Fehlerquelle in der Klausur:
pnorm(z) gibt IMMER die Fläche LINKS von z.
Wenn die Aufgabe “größer als”, “mehr als”, “oberhalb von” fragt → du brauchst 1 − pnorm(z)!

Checkliste:
• “weniger als z” / “höchstens z” / “unter z” → pnorm(z) direkt
• “mehr als z” / “mindestens z” / “über z” → 1 − pnorm(z)

Zusammenfassung: Alle Formeln auf einen Blick

Gesucht Formel Beispiel (z = 1.5)
P(Z < z) pnorm(z) pnorm(1.5) = 0.9332
P(Z > z) 1 − pnorm(z) 1 − 0.9332 = 0.0668
P(−a < Z < a) 2 · pnorm(a) − 1 2(0.9332) − 1 = 0.8664
P(Z < −z) 1 − pnorm(z) 1 − 0.9332 = 0.0668
P(a < Z < b) pnorm(b) − pnorm(a) pnorm(2) − pnorm(0.5) = 0.2857
Quick Check

Berechne P(Z > −0.5). Tipp: Nutze die Symmetrie!

P(Z > −0.5) = P(Z < 0.5) = pnorm(0.5) = 0.6915
(Weil die Normalverteilung symmetrisch ist: die Fläche rechts von −0.5 ist gleich der Fläche links von +0.5)
Part 2 — Varianz von X̄ und CLT
Concept

6. Warum interessiert uns X̄?

Das Grundproblem der Statistik

Stell dir vor, du willst das Durchschnittsgewicht aller TUM-Studenten wissen. Es gibt ca. 50.000 Studenten. Du kannst nicht alle auf die Waage stellen.

Also nimmst du eine Stichprobe: Du fragst z.B. n = 50 zufällige Studenten nach ihrem Gewicht und berechnest den Durchschnitt.

Sample Mean (Stichprobenmittelwert) X̄ = (X₁ + X₂ + ... + Xₙ) / n

Das Problem: Jede Stichprobe ist anders!

Wenn du morgen andere 50 Studenten fragst, bekommst du einen anderen Durchschnitt. Und übermorgen wieder einen anderen.

Gedankenexperiment

Stell dir vor, du wiederholst das Experiment 5 mal (jedes Mal 50 zufällige Studenten):

  • Stichprobe 1: X̄ = 73.2 kg
  • Stichprobe 2: X̄ = 75.8 kg
  • Stichprobe 3: X̄ = 71.9 kg
  • Stichprobe 4: X̄ = 74.5 kg
  • Stichprobe 5: X̄ = 76.1 kg

X̄ schwankt! Es ist selbst eine Zufallsvariable.

Kernfrage: Wie stark schwankt X̄ von Stichprobe zu Stichprobe? Das ist Var(X̄) — die Varianz des Stichprobenmittelwerts. Je kleiner Var(X̄), desto präziser ist deine Schätzung.

Intuition: Mehr Daten = weniger Schwankung

Wenn du nur n = 2 Leute fragst, schwankt X̄ wild hin und her. Bei n = 1000 ist X̄ fast immer nah am echten μ. Das formalisieren wir jetzt.

Denkfrage

Was passiert mit der Schwankung von X̄, wenn n immer größer wird? Was passiert bei n → ∞?

Die Schwankung wird immer kleiner! Bei n → ∞ geht Var(X̄) → 0.
Das heißt: Mit unendlich vielen Daten würdest du den echten Mittelwert μ exakt treffen.
Concept

7. Die zwei goldenen Regeln der Varianz

Um Var(X̄) herzuleiten, brauchst du genau zwei Regeln. Lern die und du kannst alles.

Regel 1: Konstante quadrieren und rausziehen

Variance Rule 1: Scaling Var(a · X) = a² · Var(X)

Warum a² und nicht a? Weil Varianz in quadrierten Einheiten misst. Wenn du alle Werte verdoppelst, verdoppelt sich die Streuung (SD), aber die Varianz vervierfacht sich.

Beispiele

Var(X) = 4. Dann:

  • Var(3X) = 3² · 4 = 9 · 4 = 36
  • Var(X/2) = (1/2)² · 4 = 1/4 · 4 = 1
  • Var(10X) = 10² · 4 = 100 · 4 = 400
ACHTUNG: Konstante ADDIEREN ändert die Varianz NICHT!
Var(X + 5) = Var(X)   (nicht Var(X) + 5, nicht Var(X) + 25!)

Logisch: Wenn du zu allen Werten 5 addierst, verschiebst du alles nach rechts, aber die Streuung bleibt gleich.
Allgemein: Linear Transformation Var(aX + b) = a² · Var(X)     (das b fällt weg!)

Regel 2: Varianz einer Summe (bei Unabhängigkeit)

Variance Rule 2: Addition (independent) Var(X + Y) = Var(X) + Var(Y)     wenn X, Y unabhängig

Varianzen addieren sich immer — sogar bei Subtraktion!

Beispiele

Var(X) = 4, Var(Y) = 9, X und Y unabhängig. Dann:

  • Var(X + Y) = 4 + 9 = 13
  • Var(X − Y) = 4 + 9 = 13   (Ja, auch bei Minus addiert sich die Varianz!)

Warum Var(X − Y) = Var(X) + Var(Y)? Weil X − Y = X + (−1)·Y, und Var(−Y) = (−1)² · Var(Y) = Var(Y).

Wenn NICHT unabhängig:
Var(X + Y) = Var(X) + Var(Y) + 2 · Cov(X, Y)
Die Covarianz kann positiv oder negativ sein! Aber in der Klausur sind die Xi fast immer unabhängig.

Beide Regeln kombiniert

Beispiel: Var(2X + 3Y)

Var(X) = 4, Var(Y) = 9, unabhängig.

Schritt 1: Aufteilen
Var(2X + 3Y) = Var(2X) + Var(3Y)   (Regel 2, weil unabhängig)
Schritt 2: Konstanten rausziehen
= 2² · Var(X) + 3² · Var(Y)   (Regel 1)
Schritt 3: Einsetzen
= 4 · 4 + 9 · 9 = 16 + 81 = 97
Übe selbst

Var(X) = 4, Var(Y) = 9, unabhängig. Berechne Var(X − 2Y + 7).

Var(X − 2Y + 7)
= Var(X) + Var(−2Y) + 0   (die +7 fällt weg!)
= Var(X) + (−2)² · Var(Y)
= 4 + 4 · 9
= 4 + 36 = 40
Concept

8. Der Beweis — Var(X̄) = σ²/n

Das ist DER Beweis, der in der Klausur kommt (war z.B. in der 2023er Klausur!). Jeder einzelne Schritt erklärt:

Voraussetzungen: X₁, X₂, ..., Xₙ sind unabhängig und identisch verteilt (i.i.d.) mit E(Xᵢ) = μ und Var(Xᵢ) = σ² für alle i.
Schritt 1: X̄ aufschreiben
X̄ = (X₁ + X₂ + ... + Xₙ) / n

Was tun wir hier? Einfach die Definition von X̄ hinschreiben.
Schritt 2: Das 1/n als Faktor schreiben
X̄ = (1/n) · (X₁ + X₂ + ... + Xₙ)

Warum? Weil 1/n eine Konstante ist — das brauchen wir für Regel 1!
Schritt 3: Regel 1 anwenden — Konstante rausziehen
Var(X̄) = Var((1/n) · (X₁ + X₂ + ... + Xₙ))
= (1/n)² · Var(X₁ + X₂ + ... + Xₙ)

Was passiert? Die Konstante 1/n wird quadriert und kommt vor die Var.
Schritt 4: Regel 2 anwenden — Summe aufspalten (Unabhängigkeit!)
= (1/n²) · [Var(X₁) + Var(X₂) + ... + Var(Xₙ)]

Was passiert? Weil die Xᵢ unabhängig sind, dürfen wir die Varianz der Summe als Summe der Varianzen schreiben. Ohne Unabhängigkeit geht das nicht!
Schritt 5: Alle Var(Xᵢ) sind gleich σ²
= (1/n²) · [σ² + σ² + ... + σ²]

Was passiert? Weil alle Xᵢ identisch verteilt sind, hat jedes die gleiche Varianz σ².
Schritt 6: Zählen — es gibt n Stück
= (1/n²) · n · σ²

Was passiert? Wir haben n mal σ² addiert, das ergibt n·σ².
Schritt 7: Kürzen!
= n · σ² / n² = σ² / n
Ergebnis Var(X̄) = σ² / n
Was sagt uns das?
• Mehr Daten (größeres n) → kleinere Varianz → präzisere Schätzung.
• Das ist DER Kern der Statistik: Mit mehr Daten wird deine Schätzung besser!
• Aber: Verdoppelung von n halbiert die Varianz nur — für halben SE brauchst du 4x so viele Daten.

Standard Error (SE)

Der Standard Error ist einfach die Standardabweichung von X̄:

Standard Error SE(X̄) = SD(X̄) = √(Var(X̄)) = √(σ²/n) = σ / √n
Quick Check

σ = 20. Wie groß ist der Standard Error bei n = 100? Und bei n = 400?

Bei n = 100: SE = 20/√100 = 20/10 = 2
Bei n = 400: SE = 20/√400 = 20/20 = 1

4x so viele Daten → nur halb so großer SE. Das ist die √n-Regel!
Concept

9. Der Central Limit Theorem (CLT)

Das “Wunder” der Statistik

Der CLT ist wahrscheinlich das wichtigste Theorem der gesamten Statistik. Er sagt:

Egal wie die Einzeldaten verteilt sind — der Stichprobenmittelwert X̄ ist annähernd normalverteilt, wenn n groß genug ist!
Central Limit Theorem X̄n ~ N(μ, σ²/n)    approximately, for large n

Das heißt konkret:

  • Der Erwartungswert von X̄ ist μ (der Populationsmittelwert)
  • Die Varianz von X̄ ist σ²/n (wird kleiner mit mehr Daten)
  • Die Form der Verteilung von X̄ ist annähernd eine Glockenkurve

Warum ist das so krass?

Denk mal an einen Würfel. Ein einzelner Wurf ist gleichverteilt (jede Zahl 1-6 gleich wahrscheinlich) — das sieht überhaupt nicht wie eine Glockenkurve aus!

Würfel-Beispiel
  • 1 Würfel: gleichverteilt, flach — keine Glocke
  • Durchschnitt von 2 Würfeln: schon ein bisschen hügelig
  • Durchschnitt von 5 Würfeln: sieht schon glockenförmig aus
  • Durchschnitt von 50 Würfeln: fast perfekte Normalverteilung!

Das ist der CLT in Aktion. Die Einzelwerte müssen nicht normalverteilt sein — der Durchschnitt wird es trotzdem.

Bedingungen für den CLT

  1. Unabhängigkeit: Die Beobachtungen müssen unabhängig sein (Zufallsstichprobe)
  2. Ausreichend großes n: Faustregel: n ≥ 30
Sonderfall: Wenn die Population selbst normalverteilt ist, dann ist X̄ für jedes n exakt normalverteilt — nicht nur approximativ. Der CLT wird dann gar nicht gebraucht.
Check dein Verständnis

Einkommen ist typischerweise rechtsschief verteilt (viele verdienen wenig, wenige verdienen viel). Ist der Durchschnitt von 100 Einkommenswerten trotzdem normalverteilt?

Ja! Der CLT sagt: Egal wie die Einzeldaten verteilt sind — bei n = 100 ist X̄ annähernd normalverteilt. n = 100 ≥ 30, also ist die Bedingung erfüllt.
Worked Examples

10. CLT Anwendung — Typische Klausuraufgabe

Jetzt kombinieren wir alles: Z-Score + Var(X̄) + CLT. Das ist genau der Aufgabentyp, der in der Klausur kommt.

Kochrezept für CLT-Aufgaben:
1. E(X̄) = μ aufschreiben
2. SD(X̄) = σ / √n berechnen
3. Z-Score berechnen: Z = (X̄ − μ) / SD(X̄)
4. pnorm nachschlagen und ggf. 1 − pnorm rechnen
Beispiel 1: IQ-Test

Aufgabe: IQ hat μ = 100, σ = 16. Stichprobe n = 64. Berechne P(X̄ > 104).

Schritt 1: E(X̄) = μ
E(X̄) = 100
Schritt 2: SD(X̄) berechnen
SD(X̄) = σ / √n = 16 / √64 = 16 / 8 = 2
Schritt 3: Z-Score berechnen
Z = (104 − 100) / 2 = 4 / 2 = 2
Schritt 4: Wahrscheinlichkeit
P(X̄ > 104) = P(Z > 2) = 1 − pnorm(2) = 1 − 0.9772 = 0.0228 ≈ 2.3%

Interpretation: Es ist sehr unwahrscheinlich (nur 2.3%), dass der Durchschnitts-IQ einer Stichprobe von 64 Personen über 104 liegt.

Häufiger Fehler: Den Standard Error vergessen und stattdessen σ = 16 als Nenner nehmen!
Falsch: Z = (104 − 100) / 16 = 0.25 → völlig anderes Ergebnis!
Bei X̄ immer σ/√n verwenden, nicht σ!
Beispiel 2: Paketgewichte

Aufgabe: Pakete wiegen im Schnitt μ = 2.5 kg mit σ = 0.8 kg. Ein Paketbote liefert n = 40 Pakete. Berechne P(X̄ < 2.3).

Schritt 1: E(X̄)
E(X̄) = μ = 2.5 kg
Schritt 2: SD(X̄)
SD(X̄) = 0.8 / √40 = 0.8 / 6.3246 = 0.1265
Schritt 3: Z-Score
Z = (2.3 − 2.5) / 0.1265 = −0.2 / 0.1265 = −1.581
Schritt 4: Wahrscheinlichkeit
P(X̄ < 2.3) = P(Z < −1.581) = pnorm(−1.581) = 0.0569 ≈ 5.7%

Hier brauchen wir kein “1 minus”, weil die Frage nach “kleiner als” fragt — pnorm gibt direkt die linke Fläche.

Merke den Unterschied:
• P(X > 104) bei einem Einzelwert: Z = (104−100)/16 = 0.25 → P = 0.4013 (40%)
• P(X̄ > 104) bei einem Durchschnitt von 64: Z = (104−100)/2 = 2 → P = 0.0228 (2.3%)

Für eine einzelne Person ist IQ 104 nichts Besonderes. Aber der Durchschnitt von 64 Personen bei 104? Das wäre sehr ungewöhnlich!

Exercise 5 — Variance Rules
2 pts
Var(X) = 9, Var(Y) = 16. X und Y sind unabhängig. Berechne: a) Var(2X) b) Var(X + Y) c) Var(X − Y) d) Var(3X + 2Y)
a) Var(2X)
= 2² · Var(X) = 4 · 9 = 36

b) Var(X + Y)
= Var(X) + Var(Y) = 9 + 16 = 25   (weil unabhängig)

c) Var(X − Y)
= Var(X) + Var(−Y) = Var(X) + (−1)² · Var(Y) = 9 + 16 = 25
Beachte: Var(X−Y) = Var(X+Y)! Varianzen addieren sich immer.

d) Var(3X + 2Y)
= Var(3X) + Var(2Y)   (Regel 2, unabhängig)
= 3² · Var(X) + 2² · Var(Y)   (Regel 1)
= 9 · 9 + 4 · 16
= 81 + 64 = 145

Exercise 6 — Var(X̄) Beweis (Klausur 2023!)
3 pts
X₁, X₂, ..., Xₙ seien n unabhängige Zufallsvariablen mit E(Xᵢ) = μ und Var(Xᵢ) = σ² für alle i. Schreibe den vollständigen Beweis auf, dass Var(X̄ₙ) = σ²/n. Nenne bei JEDEM Schritt, welche Regel du verwendest.
Schritt 1: Definition
X̄ₙ = (1/n) · (X₁ + X₂ + ... + Xₙ)

Schritt 2: Var der Linearkombination — Konstante rausziehen
Var(X̄ₙ) = Var((1/n) · ∑Xᵢ) = (1/n)² · Var(∑Xᵢ)
Verwendet: Var(c·Y) = c² · Var(Y) mit c = 1/n

Schritt 3: Varianz der Summe aufspalten
= (1/n²) · [Var(X₁) + Var(X₂) + ... + Var(Xₙ)]
Verwendet: Var(X+Y) = Var(X) + Var(Y) für unabhängige Variablen

Schritt 4: Identische Verteilung einsetzen
= (1/n²) · [σ² + σ² + ... + σ²] = (1/n²) · nσ²
Verwendet: Var(Xᵢ) = σ² für alle i (identisch verteilt)

Schritt 5: Vereinfachen
= nσ² / n² = σ² / n     ■

Wichtig für volle Punktzahl:
• Unabhängigkeit muss explizit erwähnt werden bei Schritt 3
• Identische Verteilung muss explizit erwähnt werden bei Schritt 4
• Ohne diese Begründungen gibt es Punktabzug!

Exercise 8 — Combined: Coffee Shop
3 pts
Ein Café füllt Kaffeetassen mit μ = 350ml und σ = 15ml. Du nimmst eine Stichprobe von 25 Tassen. a) Was ist SD(X̄)? b) P(X̄ > 355)? c) P(345 < X̄ < 355)?
SE = 15 / sqrt(25) # b) 1 - pnorm((355 - 350) / SE) # c) pnorm((355 - 350) / SE) - pnorm((345 - 350) / SE)
a) SD(X̄)
SD(X̄) = σ / √n = 15 / √25 = 15 / 5 = 3 ml

b) P(X̄ > 355)
Z = (355 − 350) / 3 = 5 / 3 = 1.667
P(X̄ > 355) = 1 − pnorm(1.667) = 1 − 0.9525 = 0.0475 ≈ 4.75%

c) P(345 < X̄ < 355)
Zunten = (345 − 350) / 3 = −5/3 = −1.667
Zoben = (355 − 350) / 3 = 5/3 = 1.667

P = pnorm(1.667) − pnorm(−1.667)
= 0.9525 − 0.0475
= 0.9050 ≈ 90.5%

Symmetrie-Shortcut: 2 · pnorm(1.667) − 1 = 2(0.9525) − 1 = 0.9050 ✔

Zusammenfassung: In 90.5% der Fälle liegt der Durchschnitt von 25 Tassen zwischen 345ml und 355ml — also ziemlich nah an den versprochenen 350ml.
