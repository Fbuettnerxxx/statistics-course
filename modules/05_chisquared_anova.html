Module 5: Chi-Squared & ANOVA

Two big topics — together worth ~15 points on the exam
Week 1 • Day 5
Concept

1. Chi-Squared Goodness of Fit (GOF)

Tests whether an observed distribution matches a hypothesized one. "Do these data follow the distribution we expect?"

Hypotheses
H0: The distribution follows the hypothesized proportions (p1, p2, ..., pk)
HA: The distribution does not follow the hypothesized proportions
Expected Count
Expectedi = n × p0,i
Chi-Squared Statistic
χ² = Σ (observedi − expectedi)² / expectedi
Degrees of Freedom
df = k − 1     (k = number of categories)

Conditions

  1. Independence: sampled observations are independent (random sample, n < 10% of population)
  2. Sample size: each expected count ≥ 5 (not observed — expected!)

R Code

chisq.test(table(data$variable), p = c(0.22, 0.18, 0.37, 0.23))

Infer Workflow (simulation-based GOF)

# Step 1: Generate the null distribution null_dist <- data |> specify(response = variable) |> hypothesize(null = "point", p = c("cat1" = 0.22, "cat2" = 0.18, "cat3" = 0.37, "cat4" = 0.23)) |> generate(reps = 1000, type = "draw") |> calculate(stat = "Chisq") # Step 2: Compute the observed test statistic obs_stat <- data |> specify(response = variable) |> hypothesize(null = "point", p = c("cat1" = 0.22, "cat2" = 0.18, "cat3" = 0.37, "cat4" = 0.23)) |> calculate(stat = "Chisq") # Step 3: Get the p-value get_p_value(null_dist, obs_stat, direction = "greater")
Reading the p-value from simulation: The p-value equals the proportion of simulated χ² statistics that are ≥ the observed χ². In formula: p = (number of simulated stats ≥ observed) / total reps. Since χ² tests are always right-tailed, we always use direction = "greater".
GOF = one variable, testing against known proportions.
The χ² statistic is always ≥ 0. Large values → more evidence against H0. The test checks all categories simultaneously.
Concept

2. Chi-Squared Test of Independence

Tests whether two categorical variables are associated or independent. Uses a contingency table.

Hypotheses
H0: The two variables are independent (no association)
HA: The two variables are not independent (there is an association)
Expected Count (under independence)
Expected = (row total × column total) / table total
Degrees of Freedom
df = (r − 1)(c − 1)     (r = rows, c = columns)

The test statistic formula is the same: χ² = Σ (O − E)² / E, summing over all cells.

R Code

chisq.test(data$var1, data$var2)

Infer Workflow (simulation-based independence)

# Step 1: Generate the null distribution null_dist <- data |> specify(formula = response ~ explanatory) |> hypothesize(null = "independence") |> generate(reps = 1000, type = "permute") |> calculate(stat = "Chisq") # Step 2: Compute the observed statistic obs_stat <- data |> specify(formula = response ~ explanatory) |> calculate(stat = "Chisq") # Step 3: Get the p-value get_p_value(null_dist, obs_stat, direction = "greater")
GOF Test One variable vs. hypothesized distribution
df = k − 1
Uses p = c(...)
null = "point", type = "draw"
Independence Test Two variables vs. each other
df = (r−1)(c−1)
Uses formula = y ~ x
null = "independence", type = "permute"
Concept

3. Interpreting Chi-Squared Results

The χ² distribution is right-skewed and always ≥ 0. Larger χ² = stronger evidence against H0.

  • Large χ² → small p-value → reject H0 → evidence of association / poor fit
  • Small χ² → large p-value → fail to reject H0 → no evidence against the null

When Expected Counts Are Close to Observed Counts

Exam shortcut (2023 retake pattern): If the expected counts are very close to (or nearly equal to) the observed counts, then every term (O − E)²/E is tiny, so the total χ² will be small. A small χ² means a large p-value, so you fail to reject H0.

You can state this reasoning without doing the full calculation: "The expected and observed counts are very similar, so χ² will be close to zero, giving a large p-value. We fail to reject H0."

Bounding p-values with qchisq()

On the exam, you often need to bound the p-value using chi-squared quantiles rather than compute it exactly:

qchisq(0.90, df = 4) # = 7.78 qchisq(0.95, df = 4) # = 9.49 qchisq(0.99, df = 4) # = 13.28
How to read the table:
qchisq(1 − α, df) gives the critical value.
If your χ² < qchisq(0.90, df) → p > 0.10
If qchisq(0.90, df) < χ² < qchisq(0.95, df) → 0.05 < p < 0.10
If qchisq(0.95, df) < χ² < qchisq(0.99, df) → 0.01 < p < 0.05
If χ² > qchisq(0.99, df) → p < 0.01

Reading p-values from Simulation Plots

The 0.9 quantile line trick (common exam question):
A simulation plot often shows a dotted vertical line at the 0.9 quantile of the null distribution, meaning 90% of simulated χ² values fall to its left and 10% fall to its right.

• If the observed statistic is to the LEFT of the 0.9 quantile line → the observed value is not in the most extreme 10% → p-value > 0.10 → fail to reject at α = 0.10
• If the observed statistic is to the RIGHT of the 0.9 quantile line → p-value < 0.10 → reject at α = 0.10

General rule: If observed is LEFT of the (1 − α) quantile line, then p > α. If RIGHT, then p < α.
Common mistake: χ² tests only tell you IF there is an association — they do NOT tell you the direction or which cells drive the association. Look at observed vs. expected to interpret further.
Concept

4. ANOVA — Comparing Multiple Group Means

ANOVA (Analysis of Variance) tests whether the means of 3 or more groups are all equal. It generalizes the two-sample t-test.

Hypotheses
H0: μ1 = μ2 = ... = μk    (all group means are equal)
HA: At least one group mean differs
F-Statistic
F = MSG / MSE = (SSG / dfG) / (SSE / dfE)
Sum of Squares Decomposition
SST = SSG + SSE
Total variability = Between-group variability + Within-group variability
Degrees of Freedom
dfG = k − 1     dfE = n − k     dfT = n − 1
(k = number of groups, n = total observations)

The ANOVA Table (memorize this!)

SourceDfSum SqMean SqF valuePr(>F)
Groupk − 1SSGMSG = SSG/(k−1)MSG/MSEp-value
Residualsn − kSSEMSE = SSE/(n−k)
Key exam trick — finding SST from var(): If the exam gives you var(data) * (n-1), that equals SST directly, because SST = Σ(xi − x̄)² = sample variance × (n − 1). From there: SSE = SST − SSG.

R² in ANOVA

Proportion of Variability Explained
R²ANOVA = SSG / SST
Interpreting R² (exam answer template):
"[R² × 100]% of the variability in [response variable] is explained by [grouping variable]."

Example: R² = 7.36/11.46 = 0.642 → "64.2% of the variability in nitrogen concentration is explained by the year of measurement."

Conditions

  1. Independence: observations within and across groups are independent
  2. Approximate normality: data within each group are nearly normal (check with QQ-plots/histograms)
  3. Constant variance: variability across groups is roughly equal (largest SD / smallest SD < 2)
Concept

5. Post-hoc: Bonferroni Correction

After rejecting the ANOVA H0, we know at least one mean differs — but which ones? Use pairwise t-tests with Bonferroni correction to control the overall Type I error rate.

Number of Pairwise Comparisons
K = C(k, 2) = k(k − 1) / 2
Bonferroni-adjusted Significance Level
α* = α / K
Adjusted p-values (alternative approach)
padj = K × poriginal     (cap at 1.0)
Example: k = 3 groups at α = 0.05
K = C(3,2) = 3 pairwise comparisons
α* = 0.05 / 3 = 0.0167

If a pairwise t-test gives p = 0.03:
Method 1: Compare p to α*: 0.03 > 0.0167 → do NOT reject
Method 2: padj = 3 × 0.03 = 0.09 > 0.05 → do NOT reject
Both methods give the same conclusion.
Example: k = 4 groups at α = 0.05
K = C(4,2) = 6 pairwise comparisons
α* = 0.05 / 6 = 0.00833
Two equivalent ways to apply Bonferroni:
Way 1 (adjust α): Compare each raw p-value to α* = α/K
Way 2 (adjust p): Multiply each raw p-value by K, then compare to original α
Both always give the same reject/fail-to-reject decision.
Only do post-hoc tests if you rejected the ANOVA H0!
If ANOVA fails to reject, you stop. No pairwise tests needed.
Worked Example

6. ANOVA Table Walkthrough — Filling in Missing Values

This type of question appears on EVERY TUM stats exam. You are given a partial ANOVA table and must fill in the blanks using the relationships between SST, SSG, SSE, df, MSG, MSE, and F.

Problem Setup (2022 Exam):
3 groups (k=3), 6 observations each (n=18 total).
Given: SSG = 7.36, F = 13.47
Hint: var(data) × 17 = 11.46
Step 1: Degrees of freedom
dfG = k − 1 = 3 − 1 = 2
dfE = n − k = 18 − 3 = 15
dfT = n − 1 = 17    (sanity check: 2 + 15 = 17 ✓)
Step 2: Use the hint to find SST
SST = var(data) × (n − 1) = var(data) × 17 = 11.46
This works because SST = Σ(xi − x̄)² = sample variance × (n−1)
Step 3: Find SSE
SSE = SST − SSG = 11.46 − 7.36 = 4.10
Step 4: Compute Mean Squares
MSG = SSG / dfG = 7.36 / 2 = 3.68
MSE = SSE / dfE = 4.10 / 15 = 0.2733
Step 5: Verify F
F = MSG / MSE = 3.68 / 0.2733 = 13.47 ✓ (matches given value)

R² = SSG/SST = 7.36/11.46 = 0.642 = 64.2% of variability explained by year.

The key exam trick: The hint "var(data) × (n−1)" always gives you SST. From there: SSE = SST − SSG. Once you have all sums of squares and df, the rest (MSG, MSE, F) follows mechanically.
Concept

7. Simulation-Based Chi-Squared Testing (Exam Detail)

The infer package lets you test hypotheses via simulation instead of relying on the theoretical χ² distribution. The exam frequently asks you to interpret simulation output or fill in the code.

GOF Simulation Workflow

null_dist <- data |> specify(response = variable) |> hypothesize(null = "point", p = c("A" = 0.25, "B" = 0.25, "C" = 0.50)) |> generate(reps = 1000, type = "draw") |> calculate(stat = "Chisq")
  • specify(response = ...): one variable only (GOF = one variable)
  • null = "point": testing against a fixed set of proportions
  • type = "draw": each rep draws n observations from the hypothesized distribution

Independence Simulation Workflow

null_dist <- data |> specify(formula = y ~ x) |> hypothesize(null = "independence") |> generate(reps = 1000, type = "permute") |> calculate(stat = "Chisq")
  • specify(formula = y ~ x): two variables (response ~ explanatory)
  • null = "independence": testing if the variables are unrelated
  • type = "permute": each rep shuffles the explanatory variable labels to break any real association

Computing the p-value from Simulation

Simulation p-value
p-value = (number of simulated χ² values ≥ observed χ²) / total reps
Worked example:
Suppose you ran 1000 simulations and the observed χ² = 8.5. If 23 of the 1000 simulated χ² values are ≥ 8.5, then:
p-value = 23/1000 = 0.023
Since 0.023 < 0.05, reject H0.

The Quantile Line Trick (visual reasoning)

How to read simulation plots on the exam:

The plot shows a histogram of the null distribution. Two vertical lines are drawn:
• Solid line = observed χ² statistic
• Dotted line = the (1 − α) quantile of the null distribution (e.g., the 0.9 quantile for α = 0.10)

Decision rule:
• Observed line is LEFT of dotted line → observed is NOT extreme → p > α → fail to reject
• Observed line is RIGHT of dotted line → observed IS extreme → p < α → reject H0

Why this works: The dotted line at the 0.9 quantile means 90% of null values are to its left and 10% are to its right. If your observed statistic is also to its left, then more than 10% of null values exceed the observed → p > 0.10.
Cheat Sheet

Module 5 Summary

Chi-Squared Tests

  1. GOF: one variable vs. hypothesized proportions. χ² = Σ(O−E)²/E, df = k−1
  2. Independence: two variables in a contingency table. Expected = (row×col)/total, df = (r−1)(c−1)
  3. Conditions (both): independence + all expected counts ≥ 5
  4. Large χ² → small p → reject H0
  5. Small χ² shortcut: if expected ≈ observed for all cells, χ² ≈ 0 → fail to reject
  6. Use qchisq() to bound p-values on the exam
  7. Infer: GOF uses null="point", type="draw". Independence uses null="independence", type="permute"
  8. Simulation p-value: count(simulated ≥ observed) / reps. Quantile line: observed LEFT of (1−α) quantile → p > α

ANOVA

  1. H0: all group means equal. HA: at least one differs
  2. F = MSG/MSE. dfG = k−1, dfE = n−k
  3. SST = SSG + SSE. Use var × (n−1) = SST
  4. R² = SSG/SST = proportion of variability explained. State as: "XX% of the variability in [response] is explained by [group]"
  5. Conditions: independence, normality within groups, constant variance
  6. Bonferroni: α* = α/K where K = C(k,2) = k(k−1)/2. Or equivalently: padj = K × praw. Only after rejecting H0

b) Reject H0?

Exercise 6 — Independence Test with Simulation
3 pts • 2023 Exam
A simulated null distribution of the χ² statistic was generated for a test of independence. The observed test statistic is marked with a solid vertical line on the plot. A dotted vertical line shows the 0.9 quantile of the null distribution (i.e., 90% of simulated statistics fall to the left of this line). The observed χ² statistic falls to the LEFT of the dotted line. Can you reject H0 at the α = 0.10 significance level? Explain your reasoning.
Key reasoning:

The dotted line marks the 0.9 quantile of the null distribution. This means 90% of simulated χ² values (under H0) fall below this line, and 10% fall above it.

Since the observed test statistic falls to the LEFT of the 0.9 quantile line, the observed value is NOT among the most extreme 10% of values under the null.

p-value: The p-value is the proportion of null distribution values ≥ the observed statistic. Since the observed value is below the 90th percentile, more than 10% of simulated values are at least as extreme as the observed statistic.
Therefore: p-value > 0.10

Conclusion: Since p-value > 0.10 = α, we cannot reject H0 at the α = 0.10 level. There is insufficient evidence to conclude that the two variables are associated. The observed χ² value is consistent with what we would expect under independence.

Visual rule of thumb: For a right-tailed χ² test, if your observed statistic is to the LEFT of the (1 − α) quantile line, then p > α and you fail to reject. If it is to the RIGHT, then p < α and you reject.

Exercise 7 — Expected Close to Observed (Quick Reasoning)
2 pts • 2023 Retake Exam
A survey asks 200 people about their preferred mode of transport (car, bike, bus, train). The hypothesized distribution is 25% each. The observed counts are: Car: 52, Bike: 48, Bus: 51, Train: 49 Without computing the exact χ² statistic, explain whether we would reject or fail to reject H0 at α = 0.05.
Expected counts under H0:
Each category: 200 × 0.25 = 50

Observed counts: 52, 48, 51, 49

Key reasoning: The observed counts (52, 48, 51, 49) are extremely close to the expected counts (50, 50, 50, 50). Each difference |O − E| is at most 2.

Since χ² = Σ(O − E)²/E, and each term (O − E)²/E will be very small (e.g., (52−50)²/50 = 4/50 = 0.08), the total χ² statistic will be close to zero.

Verification: χ² = (2² + 2² + 1² + 1²)/50 = (4 + 4 + 1 + 1)/50 = 10/50 = 0.20
With df = 4 − 1 = 3, even qchisq(0.50, 3) = 2.37, so 0.20 is far below it.
p-value > 0.50, which is much greater than 0.05.

Conclusion: We fail to reject H0. The data are fully consistent with the hypothesized uniform distribution. When expected and observed counts are very similar, there is no evidence against H0.
