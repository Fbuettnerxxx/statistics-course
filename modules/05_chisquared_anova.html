Module 5: Chi-Squared & ANOVA

Two big topics — together worth ~15 points on the exam
Week 1 • Day 5
Concept

1. Chi-Squared Goodness of Fit (GOF)

Tests whether an observed distribution matches a hypothesized one. "Do the data fit the expected proportions?"

Hypotheses
H₀: The distribution follows the hypothesized proportions (p₁, p₂, ..., pₖ)
Hᴀ: The distribution does NOT follow the hypothesized proportions
Expected Counts
Expectedi = n × p₀i
Chi-Squared Test Statistic
χ² = Σ (observedi − expectedi)² / expectedi
Degrees of Freedom
df = k − 1     (k = number of categories)

Conditions

  1. Independence: sampled observations must be independent (random sample, n < 10% of population)
  2. Sample size: all expected counts ≥ 5

R Code

chisq.test(table(data$variable), p = c(0.22, 0.18, 0.37, 0.23))

infer Workflow (simulation-based)

null_dist <- data |> specify(response = variable) |> hypothesize(null = "point", p = c("cat1" = 0.22, "cat2" = 0.18, "cat3" = 0.37, "cat4" = 0.23)) |> generate(reps = 1000, type = "draw") |> calculate(stat = "Chisq")
Key insight: GOF compares one categorical variable to a hypothesized distribution. The test checks all categories simultaneously — you cannot split it into separate tests.
Concept

2. Chi-Squared Test of Independence

Tests whether two categorical variables are associated or independent.

Hypotheses
H₀: The two variables are independent (no association)
Hᴀ: The two variables are NOT independent (there IS association)
Expected Counts (under independence)
Expected = (row total × column total) / table total
Degrees of Freedom
df = (r − 1)(c − 1)     (r = rows, c = columns)

The test statistic uses the same formula: χ² = Σ (O − E)² / E, summed over all cells.

R Code

chisq.test(data$var1, data$var2)

infer Workflow (simulation-based)

null_dist <- data |> specify(formula = response ~ explanatory) |> hypothesize(null = "independence") |> generate(reps = 1000, type = "permute") |> calculate(stat = "Chisq")
GOF Test One variable vs. hypothesized distribution
df = k − 1
Uses p = c(...)
Independence Test Two variables vs. each other
df = (r−1)(c−1)
Uses formula = y ~ x
Concept

3. Interpreting Chi-Squared Results

The χ² distribution is right-skewed and always positive. Larger χ² values mean stronger evidence against H₀.

  • Large χ² → small p-value → reject H₀ → evidence of association / poor fit
  • Small χ² → large p-value → fail to reject H₀ → no evidence against the null

Bounding p-values with qchisq()

On the exam, you often need to bound p-values rather than compute them exactly:

qchisq(0.90, df = 4) # gives 7.78 qchisq(0.95, df = 4) # gives 9.49 qchisq(0.99, df = 4) # gives 13.28

If your χ² = 8.5 with df = 4: since 7.78 < 8.5 < 9.49, the p-value is between 0.05 and 0.10.

Exam technique: Use qchisq() values to sandwich your test statistic. The p-value = P(χ² > observed), which is 1 − CDF. So if qchisq(0.95, df) = 9.49 and your statistic is 8.5, p-value > 1 − 0.95 = 0.05.
Remember: χ² tests only tell you IF there is an association, not WHERE or HOW STRONG. For strength, look at effect size measures or compare observed vs expected cell by cell.
Concept

4. ANOVA — Comparing Multiple Group Means

ANOVA (Analysis of Variance) tests whether the means of 3 or more groups differ. It generalizes the two-sample t-test.

Hypotheses
H₀: μ₁ = μ₂ = ... = μₖ    (all group means are equal)
Hᴀ: At least one group mean differs
F Statistic
F = MSG / MSE = (SSG / dfG) / (SSE / dfE)
Sum of Squares Decomposition
SST = SSG + SSE
Total variability = Between-group + Within-group
Degrees of Freedom
dfG = k − 1     dfE = n − k     dfT = n − 1
(k = number of groups, n = total observations)

The ANOVA Table

R² in ANOVA

Proportion of Variability Explained
R² = SSG / SST

Conditions

  1. Independence: observations within and across groups are independent
  2. Approximate normality: the distribution within each group should be nearly normal (check with QQ-plots)
  3. Constant variance: the variability across groups should be roughly equal (largest SD / smallest SD < 2 is a common rule)
Exam must-know: The ANOVA table fill-in question appears on virtually every TUM stats exam. Master the relationships: MSG = SSG/dfG, MSE = SSE/dfE, F = MSG/MSE, SST = SSG + SSE, dfT = dfG + dfE.
Concept

5. Post-hoc: Bonferroni Correction

ANOVA only tells you "at least one mean differs" — it does NOT tell you which pairs differ. For that, do pairwise t-tests with a Bonferroni correction.

Number of Pairwise Comparisons
K = C(k, 2) = k(k − 1) / 2
Bonferroni-adjusted Significance Level
α* = α / K
Adjusted p-value (alternative approach)
padj = K × poriginal
Example: 3 groups at α = 0.05
K = C(3,2) = 3 pairwise comparisons
α* = 0.05 / 3 = 0.0167

If a pairwise t-test gives p = 0.03:
Method 1: 0.03 > α* = 0.0167 → do NOT reject
Method 2: padj = 3 × 0.03 = 0.09 > 0.05 → do NOT reject
Both methods give the same conclusion.
Why Bonferroni matters: Without correction, if you do K tests at α = 0.05, the probability of at least one false positive is approximately K × α. Bonferroni controls this by making each individual test more strict.
Worked Example

6. ANOVA Table Walkthrough — Step by Step

This type of question appears on every single TUM exam. You are given a partial ANOVA table and must fill in the blanks.

Problem Setup:
3 groups (k=3), 6 observations each (n=18).
Given: SSG = 7.36, F = 13.47, and the hint var(data) × 17 = 11.46
Step 1: Degrees of freedom
dfG = k − 1 = 3 − 1 = 2
dfE = n − k = 18 − 3 = 15
dfT = n − 1 = 18 − 1 = 17    (sanity check: 2 + 15 = 17 ✓)
Step 2: Use the hint to find SST
SST = var(data) × (n − 1) = var(data) × 17 = 11.46
The hint "var(data) × 17 = 11.46" directly gives us SST, because SST = Σ(xi − x̄)² = var × (n−1)
Step 3: Find SSE
SSE = SST − SSG = 11.46 − 7.36 = 4.10
Step 4: Compute Mean Squares
MSG = SSG / dfG = 7.36 / 2 = 3.68
MSE = SSE / dfE = 4.10 / 15 = 0.2733
Step 5: Verify F
F = MSG / MSE = 3.68 / 0.2733 = 13.47 ✓

Cheat Sheet

Module 5 Summary

Chi-Squared Tests

  1. GOF: H₀ = distribution matches hypothesized proportions. df = k − 1. Expected = n × p₀.
  2. Independence: H₀ = two variables are independent. df = (r−1)(c−1). Expected = (row × col) / total.
  3. χ² = Σ(O − E)²/E — same formula for both tests.
  4. Conditions: independence + all expected counts ≥ 5.
  5. Large χ² → small p → reject H₀.
  6. Use qchisq() to bound p-values on the exam.

ANOVA

  1. H₀: all group means equal. Hᴀ: at least one differs.
  2. F = MSG / MSE. dfG = k−1, dfE = n−k.
  3. SST = SSG + SSE. The hint var × (n−1) = SST.
  4. R² = SSG / SST = proportion of variability explained by groups.
  5. Conditions: independence, normality within groups, constant variance.
  6. Bonferroni: α* = α / K where K = k(k−1)/2.

Exercise 1 — Goodness of Fit Test
4 pts • 2023 Exam
The ANES survey collected data on which US census region respondents live in. The census says the population distribution should be: Northeast 22%, Midwest 18%, South 37%, West 23%. Running chisq.test(table(anes$region), p = c(0.22, 0.18, 0.37, 0.23)) in R gives: X-squared = 2.4861, df = 3, p-value = 0.4778
a) State H₀ and Hᴀ in context.
b) Check the conditions for this test.
c) State the conclusion in context at α = 0.05.
a)
H₀: The distribution of ANES respondents across census regions follows the census distribution (Northeast 22%, Midwest 18%, South 37%, West 23%).
Hᴀ: The distribution of ANES respondents across census regions does NOT follow the census distribution.

b)
1. Independence: The ANES survey uses random sampling, and the sample is less than 10% of the US population. ✓
2. Expected counts: Each expected count = n × p₀i. Since n is large and all proportions are at least 0.18, all expected counts will be well above 5. ✓

c)
Since the p-value = 0.4778 > α = 0.05, we fail to reject H₀. There is no convincing evidence that the regional distribution of ANES respondents differs from the US census distribution. The sample appears representative of the population with respect to region.

Exercise 3 — Chi-Squared p-value Bounds
2 pts
A χ² test of independence yields a test statistic of 5.0 for the relationship between socioeconomic status (3 levels: low, middle, high) and game preference (3 levels: strategy, action, puzzle). State a range for the p-value.
qchisq(0.50, df = 4) = 3.36 qchisq(0.75, df = 4) = 5.39 qchisq(0.80, df = 4) = 5.99 qchisq(0.90, df = 4) = 7.78 qchisq(0.95, df = 4) = 9.49
Step 1: Compute df
df = (r − 1)(c − 1) = (3 − 1)(3 − 1) = 4

Step 2: Locate the test statistic
qchisq(0.50, 4) = 3.36 < 5.0 < qchisq(0.75, 4) = 5.39

Step 3: Determine p-value range
The p-value = P(χ² > 5.0).
Since 5.0 is between the 50th and 75th percentile of the χ²(4) distribution:
p-value is between 1 − 0.75 = 0.25 and 1 − 0.50 = 0.50.

0.25 < p-value < 0.50

Since p > 0.05, we fail to reject H₀. No evidence of association between socioeconomic status and game preference.

Exercise 5 — ANOVA SST & F-test
3 pts • 2023 Retake Exam
Three versions of an exam (a, b, c) were given to students. The ANOVA yields: SSG = 1290, SSE = 29810, F = 3.484. a) Compute SST. b) Can H₀ (equal means across versions) be rejected at α = 0.05?
qf(0.95, df1 = 2, df2 = 161) = 3.051
a)
b)
a)
SST = SSG + SSE = 1290 + 29810 = 31100

b)
The critical value at α = 0.05 is qf(0.95, df1 = 2, df2 = 161) = 3.051.
Since F = 3.484 > 3.051, the F statistic exceeds the critical value.

Therefore, we reject H₀ at α = 0.05. There is sufficient evidence that the mean exam score differs across at least two of the three exam versions.

Note on df: k = 3 versions, so dfG = 2. From F = MSG/MSE = (1290/2)/(29810/dfE) = 3.484, we can solve dfE = 29810 / (1290/2 / 3.484) = 29810 / 185.1 = 161, confirming n = 164 students total.

Exercise 6 — Independence Test with Simulation
3 pts • 2023 Exam
A simulated null distribution for a χ² independence test is shown. The observed test statistic is marked with a solid vertical line. A dotted vertical line marks the 0.9 quantile of the null distribution (i.e., 90% of simulated values fall to the left of it). The observed test statistic falls to the LEFT of the dotted line (0.9 quantile). Can you reject H₀ at the α = 0.1 significance level? Explain your reasoning.
Key reasoning:

The dotted line marks the 0.9 quantile of the null distribution. This means 90% of simulated χ² values (under H₀) fall below this line, and 10% fall above it.

Since the observed test statistic falls to the left of the 0.9 quantile line, the observed value is not among the most extreme 10% of values.

The p-value = proportion of null distribution ≥ observed statistic. Since more than 10% of the null distribution lies to the right of the observed statistic:
p-value > 0.10 = α

Therefore, we cannot reject H₀ at α = 0.1. There is insufficient evidence to conclude that the two variables are associated.

Visual rule: For a χ² test, if your observed statistic is to the LEFT of the (1 − α) quantile line, p-value > α → fail to reject. If it's to the RIGHT, p-value < α → reject.
