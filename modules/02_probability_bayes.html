Concept

1. Probability Basics — The Three Rules You Need

Probability P(A) is always between 0 and 1. Three rules handle everything:

Rule 1: Complement
P(Ac) = 1 − P(A)
"Probability something does NOT happen = 1 minus probability it does"
Rule 2: Addition (OR)
P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
"A or B = add them, but subtract the overlap so you don't double-count"
Rule 3: Multiplication (AND, when independent)
P(A ∩ B) = P(A) · P(B)     only if A and B are independent
"A and B both happen = multiply, but ONLY if they don't influence each other"
Quick example:
P(rain) = 0.3, P(traffic jam) = 0.5, assume independent.
P(rain AND traffic jam) = 0.3 × 0.5 = 0.15
P(rain OR traffic jam) = 0.3 + 0.5 − 0.15 = 0.65
P(no rain) = 1 − 0.3 = 0.7
Concept

2. Conditional Probability — "Given that..."

The most important concept in this module. P(A|B) reads as "probability of A, given that B happened."

Conditional Probability
P(A | B) = P(A ∩ B) / P(B)

Think of it as: zoom into the world where B happened, then ask how often A also happens.

From a contingency table:
Uses BusDoesn't use BusSum
Owns car30120150
No car8020100
Sum110140250
P(Uses Bus | Owns car) = 30/150 = 0.20
P(Uses Bus | No car) = 80/100 = 0.80

These are different → bus usage depends on car ownership → not independent
Exam trick for independence from a table:
Compare P(A | B₁) vs P(A | B₂) vs P(A | B₃)...
If they're all the same → independent.
If any differ → NOT independent.
Concept

3. Independence — When Events Don't Affect Each Other

Two events A and B are independent if knowing one happened gives you no information about the other.

A and B are independent if (any of these — they're equivalent):
P(A | B) = P(A)     or     P(B | A) = P(B)     or     P(A ∩ B) = P(A) · P(B)
How to check in the exam:
Method 1: Compute P(A|B) and compare with P(A). Same? → Independent.
Method 2: Compute P(A ∩ B) and P(A) · P(B). Same? → Independent.
From a contingency table: compare conditional probabilities across columns (see above).
Common mistake: "Independent" and "mutually exclusive" are NOT the same!
Mutually exclusive: P(A ∩ B) = 0 (can't happen together)
Independent: P(A ∩ B) = P(A)·P(B) (don't influence each other)
If events are mutually exclusive AND both have P > 0, they are actually dependent!
Concept

4. Bayes' Theorem — The Star of Every Exam

Bayes lets you flip a conditional probability. You know P(B|A), but you want P(A|B).

Bayes' Theorem
P(A | B) = P(B | A) · P(A) / P(B)

The tricky part is computing P(B), the total probability. Use the Law of Total Probability:

Law of Total Probability (denominator of Bayes)
P(B) = P(B|A₁)·P(A₁) + P(B|A₂)·P(A₂) + ... + P(B|Aₖ)·P(Aₖ)
Bayes' Theorem in full (the version you need for the exam):

P(A₁ | B) = P(B|A₁)·P(A₁) / [ P(B|A₁)·P(A₁) + P(B|A₂)·P(A₂) + P(B|A₃)·P(A₃) ]
Worked Example

5. Bayes Step-by-Step — The Parking Lot Problem

This exact structure appears in every exam (with different numbers).

Problem:
A parking lot fills up due to events. Cultural events happen 33% of evenings, sporting events 22%, no events 45%. The lot fills up 21% of the time during cultural events, 75% during sporting events, and 4% when there are no events.

Jose finds the lot full. What is the probability there's a sporting event?
Step 1: Define your events
A₁ = sporting event, A₂ = cultural event, A₃ = no event
B = parking lot is full
We want: P(A₁ | B)
Step 2: Write down what you know
P(A₁) = 0.22    P(A₂) = 0.33    P(A₃) = 0.45
P(B|A₁) = 0.75    P(B|A₂) = 0.21    P(B|A₃) = 0.04
Step 3: Compute the denominator P(B)
P(B) = P(B|A₁)·P(A₁) + P(B|A₂)·P(A₂) + P(B|A₃)·P(A₃)
P(B) = 0.75×0.22 + 0.21×0.33 + 0.04×0.45
P(B) = 0.165 + 0.0693 + 0.018 = 0.2523
Step 4: Apply Bayes
P(A₁ | B) = P(B|A₁)·P(A₁) / P(B)
P(A₁ | B) = (0.75 × 0.22) / 0.2523
P(A₁ | B) = 0.165 / 0.2523 = 0.654
Always these 4 steps in the exam:
1. Define events and what you want
2. List all given probabilities
3. Compute denominator (total probability)
4. Plug into Bayes

6. Contingency Tables — Reading Probabilities

The exam gives you a table and asks you to compute probabilities from it. Here's how:

Opinion AOpinion BSum
Group 14060100
Group 27030100
Sum11090200

Three types of probability from a table:

  • Joint probability: P(Group 1 AND Opinion A) = 40/200 = 0.20 → use table total
  • Marginal probability: P(Group 1) = 100/200 = 0.50 → use row/column total
  • Conditional probability: P(Opinion A | Group 1) = 40/100 = 0.40 → use row total of the condition
Conditional = zoom into the row/column of the condition, then divide.
P(A | B) → find the row/column where B is true, then look at A within it.
Concept

7. Expected Value — The Average Outcome

Expected Value
E(X) = x₁·P(X=x₁) + x₂·P(X=x₂) + ... + xₖ·P(X=xₖ)

Multiply each outcome by its probability, then add them all up.

Card game example (exam classic):
Draw from 52 cards: Red card → win nothing. Spade → win €5. Club → win €10. Ace of clubs → win €40.

Outcomes: 0 (26 cards), 5 (13 cards), 10 (12 clubs minus ace), 40 (1 ace of clubs)
E(X) = 0×(26/52) + 5×(13/52) + 10×(12/52) + 40×(1/52)
E(X) = 0 + 1.25 + 2.308 + 0.769 = €4.33
Cheat Sheet

Module 2 Summary

  1. P(Ac) = 1 − P(A)
  2. P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
  3. P(A ∩ B) = P(A)·P(B) only if independent
  4. P(A|B) = P(A∩B) / P(B) → "zoom into B's world"
  5. Independent ↔ P(A|B) = P(A) ↔ P(A∩B) = P(A)·P(B)
  6. Bayes: flip the conditional → always 4 steps
  7. Contingency table: joint = /total, marginal = /row or col total, conditional = /condition total
  8. E(X) = sum of (outcome × probability)

Exercise 1 — Bayes (the big one)
4 pts • Exam guaranteed
A company screens job applicants with an aptitude test. 25% of applicants are actually qualified. The test correctly identifies qualified applicants 88% of the time (sensitivity). It correctly identifies unqualified applicants 82% of the time (specificity). An applicant tests positive. What is the probability they are actually qualified? Show all steps: define events, list probabilities, compute denominator, apply Bayes.
Step 1: Define events
A = applicant is qualified, Ac = unqualified
B = tests positive
Want: P(A | B)

Step 2: Given probabilities
P(A) = 0.25, P(Ac) = 0.75
P(B|A) = 0.88 (sensitivity = correctly identifies qualified)
P(Bc|Ac) = 0.82 (specificity) → so P(B|Ac) = 1 − 0.82 = 0.18

Step 3: Denominator
P(B) = P(B|A)·P(A) + P(B|Ac)·P(Ac)
P(B) = 0.88×0.25 + 0.18×0.75
P(B) = 0.22 + 0.135 = 0.355

Step 4: Bayes
P(A|B) = 0.22 / 0.355 = 0.6197 ≈ 62.0%

Key insight: Even with a pretty good test (88% / 82%), only 62% of positive results are actually qualified — because qualified applicants are rare (only 25%).

Exercise 3 — Independence Check
2 pts
Events A and B satisfy: P(A) = 0.6, P(B) = 0.3, P(A ∩ B) = 0.18. a) Are A and B independent? Justify mathematically. b) Compute P(A | B).
a)
b)
a) Check: P(A) · P(B) = 0.6 × 0.3 = 0.18
Compare with P(A ∩ B) = 0.18
Since P(A ∩ B) = P(A)·P(B), A and B are independent. ✓

b) P(A | B) = P(A ∩ B) / P(B) = 0.18 / 0.3 = 0.6
Note: this equals P(A) = 0.6, which confirms independence (knowing B doesn't change the probability of A).

A game works as follows: Roll a fair die. If you roll a 6, you win €15. If you roll a 4 or 5, you win €5. For any other number, you lose €3. a) What is the expected profit/loss per game? b) If you play 200 games, what are your expected total winnings?
a)
b)

Win €15 (roll 6): P = 1/6
Win €5 (roll 4 or 5): P = 2/6
Lose €3 (roll 1, 2, or 3): P = 3/6

E(X) = 15×(1/6) + 5×(2/6) + (−3)×(3/6)
E(X) = 2.50 + 1.667 − 1.50 = €2.667

On average you win about €2.67 per game.

b) 200 × €2.667 = €533.33

Exercise 5 — Bayes with Twins (Exam Classic)
3 pts
About 30% of human twins are identical (rest are fraternal). Identical twins are always the same sex — 50% both male, 50% both female. Among fraternal twins: 26% both male, 28% both female, 46% mixed. a) You learn your newborn twins are both boys. What is the probability they are identical? b) Are the events "twins are identical" and "both boys" independent? Justify.
a)
b)
a) Step 1: I = identical, Ic = fraternal, M = both male. Want P(I|M).

Step 2:
P(I) = 0.30, P(Ic) = 0.70
P(M|I) = 0.50, P(M|Ic) = 0.26

Step 3: Denominator
P(M) = P(M|I)·P(I) + P(M|Ic)·P(Ic)
P(M) = 0.50×0.30 + 0.26×0.70 = 0.15 + 0.182 = 0.332

Step 4: Bayes
P(I|M) = 0.15 / 0.332 = 0.4518 ≈ 45.2%

b) For independence: P(I) must equal P(I|M).
P(I) = 0.30, but P(I|M) = 0.452.
Since 0.30 ≠ 0.452, the events are NOT independent.
Knowing both twins are boys increases the probability they're identical.

Exercise 6 — Addition Rule
1 pt
P(A) = 0.45, P(B) = 0.35, P(A ∩ B) = 0.10. Compute P(A ∪ B) and P(Ac ∩ Bc).
P(A ∪ B) = P(A) + P(B) − P(A ∩ B) = 0.45 + 0.35 − 0.10 = 0.70

P(Ac ∩ Bc) = P((A ∪ B)c) = 1 − P(A ∪ B) = 1 − 0.70 = 0.30
(By De Morgan's law: "neither A nor B" = complement of "A or B")
